# Round 3 Showrunner Synthesis - 《黑绳》v1 复审汇总

## 本轮参与 agent

- 反方编剧：检查故事是否滑向俗套、规则是否自洽、观众是否会误读。
- 观众：从电影节观众、短视频观众、AI 视频创作者三类视角看懂度与观看欲。
- 导演：检查每场戏的情绪、调度、动作化表达和反熟套处理。
- 审片人：以 10 分钟电影节短片标准给分，并判断下一版优先级。

本轮结论非常一致：**《黑绳》v1 的方向成立，不需要推翻；下一版要做的是把黑绳制度、阿沅赴湖动机、安全屋因果三件事拍得更硬。**

## 总体评分与状态

当前约为 **7.5 / 10** 的电影节短片潜力稿。

它已经不是普通“献祭少女 + 湖神天罚”的低级民俗故事，因为 v1 已经有了几件高级的事：

1. 阿沅不是纯白受害者，她身边确实会发生灾祸。
2. 村民不是纯恶人，他们确实被损失、恐惧和愧疚困住。
3. 真正恐怖的不是湖神，而是“把坏运气转嫁给下一个婴儿”的制度。
4. 阿沅最后剪断的是新黑绳，不是自己的旧黑绳，这让她的行动从逃命变成断链。

但它还没有到 8.5 分以上，原因也很集中：观众第一次看时，可能会把它自动脑补成更熟悉的版本：可怜少女被献祭，死后保护恋人，恶村遭天谴。

下一版必须主动阻止这个误读。

## 必须保留

### 1. 阿沅剪断新黑绳

这是 v1 最强的动作。它说明她不是为了洗清自己，也不是为了顺从献祭，而是拒绝下一个孩子继续变成容器。

### 2. 梁嫂

梁嫂让村庄复杂起来：她受害、害怕、参与制度，但仍然心软。她不能被删，也不能变成单纯好人。

### 3. 村长的“算没淹”

这句台词是制度的冷酷账本，必须保留。它比解释湖神传说更有力。

### 4. 湖神无言

湖神不能像怪物登场，也不能像审判者。它越沉默，故事越像一套旧账在执行。

### 5. 阿澈承认怕过

阿澈不是纯爱救世主。他怕过，但仍然选择不让黑绳继续绑到下一个人身上，这比单纯救爱人更有力。

## 必须修正的三根钉子

### 1. 黑绳规则要一眼懂

目前黑绳承担太多功能：旧债、新容器、门环结、桥头反祭法、制度断链。功能都对，但观众第一次看可能跟不上。

下一版要加入一个极短的祭法公式：

> 旧的下水，新的上脚，中间不能空。

这句话不应该像解释神话，而应配合动作出现：村长把旧绳拉向湖的方向，同时把新黑绳绕成婴儿脚踝大小的空圈。观众只要看懂这个动作，后面阿沅剪新绳、系旧绳到石环才成立。

### 2. 阿沅赴湖不能像愧疚自杀

现在她剪断新黑绳后为什么还要入湖，仍然需要再钉牢。

下一版的准确含义应是：

> 她不是去赎罪，也不是去救村庄；她是带着旧黑绳下水，让转嫁到自己这里为止。

湖桥一场要把动作拍清楚：她看一眼岸边空摇篮，把旧黑绳一端穿过桥头石环打死结，另一端重新勒紧自己脚踝。村长第一次失态说“不对”。这一刻观众要明白，她在破坏祭法，不是在完成祭法。

### 3. 阿澈家安全不能像爱情祝福

门环黑结是强画面，但机制还要更硬。

第 3 场必须给一个清晰近景：阿沅把从脚踝旧绳磨断下来的唯一一小截，系在阿澈家门环上。村长撞门进来时，目光落到黑结上，停一瞬。

结尾可以保留水绕开小屋，但要避免拍成“阿沅死后回来守护恋人”。小湿脚印很美，但风险很高。更稳的替代是：门槛内侧一小滴黑水停住，黑结滴水，屋内灯未灭。阿澈不是被奖赏，而是被留下见证。

## 下一版具体改法

### 第 1 场：祠堂会议

补一个湿粮代价镜头。粮袋倒后，不只拍粮混进泥水；拍一个村民扑过去用手捧米，旁边有人伸手去抓，被母亲或梁嫂拉回。

目的：让观众理解村民恐惧有现实来源，而不是一群迷信恶人欺负小孩。

### 第 2 场：接生婆旧屋

空婴儿床不要只是恐怖陈设。让阿沅看到床栏上的旧绳痕、空脚环、褪色襁褓。

目的：说明这套制度重复过，不是阿沅一个人的悲剧。

### 第 3 场：阿澈家

门环黑结必须成为本场核心动作。阿沅用发抖的手把旧黑绳断片系在门环上，镜头要让观众记住。

阿澈的反抗也要从“我救你”升级到“不能再有下一个你”。他可以不说大话，但行为要从私奔转向断链。

### 第 4 场：黎明祠堂

加入祭法公式动作：

- 旧绳指向湖。
- 新绳绕成婴儿脚踝大小的空圈。
- 村长说：“旧的下水，新的上脚，中间不能空。”
- 阿澈指出村里没有婴儿。
- 村长的目光扫向门外、村口、避雨的陌生包袱，暗示他准备等外来孩子。
- 阿沅剪断新黑绳。

目的：让观众一眼明白，阿沅剪的不是逃生绳，而是制度的下一环。

### 第 5 场：湖边祭桥

悬念不应是“她跳不跳”，而是“她怎么系绳”。

她把旧绳一端系死在桥头石环，另一端重新系紧自己脚踝。这个动作要慢、清楚、安静。村长说“不对”，岸上没人敢立刻上桥。

目的：把赴湖从顺从献祭改成反祭法。

### 第 6 场：洪水结尾

洪水不要拍成恶村被罚。要拍成所有补救失败：

- 新黑绳泡散，缠不上任何脚踝。
- 空摇篮灌水，失去用途。
- 梁嫂或普通村民在水里救人，证明他们不是该死反派。
- 村长抱着空摇篮说“来不及了，这个村子没有新的婴儿了。”
- 阿澈屋内灯未灭，门环黑结滴水，水停在门槛。

目的：结尾不是天罚，而是制度终于找不到下一个身体。

## 对编剧 agent 的下一轮任务

下一轮不要重开世界观，不要新增角色，不要扩大场面。只做一件事：

**在 v1 基础上改出 v2 剧本，把黑绳规则、阿沅赴湖动机、安全屋因果全部动作化。**

必须完成：

1. 强化开场灾祸的现实代价。
2. 强化空婴儿床和旧绳痕，说明制度重复过。
3. 明确新黑绳是下一个脚踝。
4. 明确阿沅剪断新黑绳是在阻止下一个孩子。
5. 明确湖桥反祭法动作。
6. 弱化小湿脚印，避免爱情鬼故事误读。
7. 给梁嫂或普通村民一个最后的人性动作，避免报应爽感。

## Showrunner 判断

《黑绳》现在已经有一部 10 分钟短片的骨架。下一步不是继续发散，而是进入 v2 精修。

这轮 agent team 的价值也比较明确：四个 agent 没有把故事带向四个方向，而是在同一个关键问题上形成了交叉验证。它们共同指出：故事最危险的不是不够奇，而是核心因果不够硬。一旦因果钉牢，这个短片就能从“氛围好的民俗恐怖”往“制度寓言”走。
