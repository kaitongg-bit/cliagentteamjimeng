# Round 2 - Screenwriter Revision Notes

## 本轮采用方向

采用第一轮推荐的“献祭会议”方向，压缩为祭日前夜到清晨的一夜结构。剧本 v1 共 6 场，主要人物限定为阿沅、阿澈、村长、梁嫂四个功能清晰的人物，其他村民只作为群体压力出现。

## 结构处理

1. 祠堂开场：用阿沅无意造成的水碗、油灯、粮袋连锁事故，在前15秒建立“可怜但危险”的双信号。
2. 后巷/接生婆旧屋：通过空婴儿床、旧木牌、梁嫂被二次灾祸烫伤，补足无新婴儿和女孩负罪感。
3. 阿澈家：让恋人承认自己也怕过她，避免纯爱救世主；同时埋下门环黑结，为结尾安全屋做视觉铺垫。
4. 祠堂揭密：村长只给三句硬信息，主要靠黑绳、空摇篮、新黑绳完成制度解释。
5. 湖边献祭：阿沅不说圣母台词，而是通过剪断空摇篮里的新黑绳、改变桥上黑绳系法，让替换链条断掉。
6. 洪水结尾：献祭成功杀死阿沅，但无法保存村庄；湖神只作为水中黑影出现，不解释、不审判。

## 对 Showrunner 要求的落实

- **方向采用“献祭会议”**：第一场即为祠堂会议，第四场回到会议空间完成揭密和选择。
- **主体时间压缩在祭日前夜到天亮**：所有场景发生在雨夜、黎明前、清晨。
- **主要人物控制**：阿沅、阿澈、村长、梁嫂四人承担主要动作和台词。
- **黑绳**：作为灾祸容器、转移链条、断链动作、安全屋标记贯穿全片。
- **空摇篮**：第一场即出现，第四场成为揭密核心，第六场漂走。
- **小型连锁灾祸**：开场粮袋事故、旧屋梁嫂烫伤，两次实锤但不机械解释。
- **湖神不说话**：只在湖底黑影和水墙黑影中出现。
- **恋人屋安全**：第三场门环黑结埋伏，第六场洪水绕开小屋兑现。
- **女孩选择断链**：她看见村长准备继续给“下一个脚踝”系黑绳后，剪断新黑绳，再赴湖。
- **结尾系统崩塌**：洪水吞村不是复仇宣判，重点拍空摇篮、新黑绳无处可系、村长意识到来不及。

## 风险与后续可修方向

1. 阿沅年龄与“恋人”关系仍需 Human 定调。v1 已按童年依恋/少年守护处理，避免成人爱情质感。
2. 黑绳保护阿澈家的机制保持诗性和视觉化，没有做硬解释。若后续观众反馈“不懂”，可在湖边增加阿沅看门环黑结的回忆闪回。
3. 村长目前偏冷，下一版可增加一个短动作：他在献祭前替阿沅擦去脸上泥水，强化“执行者也有愧疚”。
4. 梁嫂作为受害/质疑村民的弧线已经成立，但她最后在洪水中的结局未单独收束。若要更悲悯，可在第六场给她扶老人或救祭器失败的画面。
5. 结尾小脚印带一点超自然余味。如果 Human 希望更冷峻现实，可删去小脚印，只保留黑结和干燥门槛。

## 下一轮建议

建议下一轮由 Script Doctor / Showrunner 重点检查三件事：

1. 黑绳规则是否足够清楚且不过度解释。
2. 阿沅最后剪断新黑绳的动作是否能明确表达“拒绝下一个婴儿”。
3. 结尾是否仍有“村民活该”的爽感，需要继续降低报应感、提高崩塌感。
