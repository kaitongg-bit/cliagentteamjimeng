# Round 1 - Screenwriter Concept Options

## 核心戏剧问题

当一个村庄真的靠牺牲一个孩子换来十年平安，而这个孩子也确实会把灾祸带给身边人时，一个爱她的人究竟是在救一个无辜者，还是在释放全村压住的坏运气？

更锋利的内核是：如果一个共同体的生存依赖于把灾难集中到一个人身上，那么这个共同体在道德上和现实上还能维持多久？

## 方向一：逃亡到湖边

### Logline

祭湖前夜，被选中的灾祸女孩被恋人从祠堂带走，两人只想逃到山路尽头，但女孩身边接连发生的小事故让追捕的村民伤亡加重；天亮前，她必须决定是继续活下去，还是主动走回湖心。

### 主角

女孩。十岁出头或刚满十岁，被村庄养成“公共灾祸容器”。她有被保护的渴望，也知道自己走到哪里，哪里就会坏掉。

### Want / Need

- Want：和恋人逃走，离开献祭命运。
- Need：承认“活下去”不能只靠别人替她否认现实，她必须亲手选择自己如何结束这套制度的最后一天。
- 弧光：从被夺走、被保护、被追捕的祭品，变成第一个清醒看见规则漏洞并主动改变结尾的人。

### 10分钟可拍性

适合一个夜晚、一条逃亡线、三个主要空间：祠堂、湿滑山路、湖边。灾祸不需要特效，可以用极小动作连锁呈现：油灯倒、绳索断、竹桥塌、追捕者摔伤、船桨裂开。信息通过追赶、受伤、村长一句关键解释完成。

前15秒钩子：祭桌上，女孩被绑住，供碗里的水无风自溢，蜡烛接连熄灭；村民同时后退一步。

### 结尾图像

女孩沉入湖心后，水面短暂平静。下一秒，远处村口的灯一盏盏灭掉，洪水越过屋脊。只有恋人被她事先推入的一间破屋还亮着灯，屋门外漂来祭祀用的新生儿襁褓，却是空的。

## 方向二：献祭会议

### Logline

村庄在祠堂召开献祭会议，女孩的恋人闯入抢人，引发一场短暂而混乱的逃离；当村长被迫说出“每十年换一个婴儿”的真相，村民、恋人和女孩都必须在同一个夜晚面对这套制度的最后一次运转。

### 主角

恋人。一个年轻、冲动、相信“她不是灾祸”的人。他的行动撕开会议的秩序，也逼出村庄一直藏着的真相。

### Want / Need

- Want：把女孩从献祭中救走，证明村民迷信而残忍。
- Need：看见事情比“坏村民害好女孩”更可怕：村庄或许真的靠她挡过灾，但这不让献祭变成正确。
- 弧光：从用爱情反抗全村的少年，变成唯一背着真相活下来的人。

### 10分钟可拍性

这是最集中、最舞台化但仍可电影化的版本。主体发生在祠堂和湖边，人物少：女孩、恋人、村长、一个质疑献祭的村民。会议本身可以通过具体损失建立村民立场：瘸腿者、烧毁的粮袋、破船牌位。恋人抢人提供动作，村长揭密提供中段转折。

前15秒钩子：村民投石表决，写着女孩名字的木牌被放进水盆，水立刻变黑；恋人从人群后方冲进来，掀翻祭桌。

### 结尾图像

洪水灌进祠堂，村长抱着空摇篮站在神像前，看见湖面上有一个巨大的影子经过。他回头，村里所有婴儿床都是空的。切到高处：恋人所在的小屋浮在水上，门槛内干燥，女孩留下的一根红绳系在门环上。

## 方向三：最后一个婴儿

### Logline

献祭日临近，村长发现村里已经没有新的婴儿可承接坏运气，却仍决定完成祭湖；女孩和恋人的逃亡迫使他暴露真相，而这次献祭从一开始就注定无法换来新的十年。

### 主角

村长。不是单纯恶人，而是旧制度最后的执行者。他知道规则、知道代价，也知道这一次少了最关键的“替换者”。

### Want / Need

- Want：完成献祭，维持村庄对秩序的信任，哪怕只是多撑一夜。
- Need：承认他守护的不是村庄，而是一套已经失效、只会继续吞人的机制。
- 弧光：从冷静主持仪式的人，变成临死前终于明白“没有新的婴儿”不是程序失败，而是整个制度的审判。

### 10分钟可拍性

适合做成更冷、更宿命的民俗恐怖。主要动作仍然清楚：准备祭品、恋人抢人、抓回、献祭、洪水。村长线通过空摇篮、登记木牌、他反复看向某户无婴的屋子来铺设，不需要解释太多神话。风险是主角偏向制度方，观众情感入口会更冷，需要让女孩和恋人的段落足够刺痛。

前15秒钩子：村长打开祖柜，里面一排写着年份的婴儿木牌，最后一格空着；门外传来女孩打碎水缸的声音，院中鸡犬惊散。

### 结尾图像

村长被洪水困在祠堂梁上，怀里抱着空襁褓。他看见湖神的黑影停在门外，像在等待下一件祭品。村长低头，襁褓里只有一块写着“十年”的旧木牌，墨迹被水泡开。

## 推荐方向

推荐方向二：献祭会议。

理由：它最适合10分钟短片体量。核心冲突集中在一个公共场域里，恋人抢人能立刻启动动作，女孩的灾祸属性可以用会议中的连锁事故呈现，村民的恐惧与愧疚也能在同一空间被看见。它保留了 Human seed 中最有力量的几件事：恋人夺走女孩、有人追问是否必然、村长揭示十年规则、女孩最终归湖、村庄仍被淹没、没有新的婴儿。

这个方向也最利于道德不稳定：观众会先站在恋人一边，随后看见村民的真实损失，再看见女孩的灾祸并非全是污名，最后仍然判断这个制度必须失败。

## 下一轮需要 Human 决策的问题

1. 女孩年龄要不要严格设定为“刚满十岁”？如果保留恋人关系，建议处理成童年依恋/少年守护，而不是成人爱情。
2. 湖神是否要被明确看见？建议只在结尾给村长一个半可见的影子，保持民俗恐怖的不确定性。
3. 女孩最后赴死是完全自愿，还是在“自愿”和“被逼到无路”之间摇摆？建议后者，更残酷也更稳定。
4. 恋人为什么所在的屋子安全？可以是女孩最后把灾祸从他身上带走，也可以是她临死前把唯一的“好运”留给他。
5. 结尾是否要明确“村里没有新生儿”的原因？建议不解释成设定，只用空摇篮、空婴儿床、村长一句“来不及了”完成。
