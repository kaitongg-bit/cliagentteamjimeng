# 第 4 场修订：增强“女明星也有错”的第二次站队

## 问题诊断

上一版第 4 场把孟青桐的免责上下文给得太早、太完整：

- 群聊里立刻出现“云帝那天录制到凌晨，别叫他了”。
- 多机位很快显示节目组 cue、她递话筒、李云帝没接。
- 颁奖词马上补出“还有云帝哥”。

结果观众还没真正站到“女明星也有问题”，就已经被剧本拉回理性判断。第二次站队失败。

本轮修订目标：让孟青桐先出现几段确实刺人的社交行为，但不把她写成反派。她可以有不体面、怯懦、随大流、为了自保不纠正误会的部分；之后再逐步证明，观众对她的“定罪”仍然被剪辑、平台标题和集体情绪放大了。

【场景4：内景 许临出租屋 / 素材库新文件夹 / 夜】

项目名闪了一下：

`female_bully_final_01`

他点开 `01_群聊不带他`。

一张聊天截图打开。群名被红框圈住：

`青桐生日局`

成员列表里没有李云帝。

`别拉他吧，他来了大家都要围着他转。`

`老板来了不像朋友局。`

`他会不会又说自己不舒服哈哈。`

最后一行是孟青桐发的。

`别这么说。`

隔了几秒。

`那就小群先定。`

许临把前一句裁掉。

屏幕里只剩：

`那就小群先定。`

甲方文字弹出：

`这个好，够准。`

许临没有回。

监看窗口里，综艺圆桌。李云帝说完一句话，低头喝水。

孟青桐和旁边女嘉宾对视。

女嘉宾用口型说了两个字：

`云底`

孟青桐先没笑。

下一秒，她低头，嘴角动了一下。

圆桌另一边有人笑出声。

李云帝抬眼。

笑声停得很快。

许临把这一段拖进时间线。

他没有加混响。

只把现场笑声提亮。

回放。

李云帝抬眼的那一帧被停住。

许临加字幕：

`他听见了。`

他看了一眼，删掉。

改成：

`他们都听见了。`

手机震动。

房东消息：

`收到钱我就不说了。`

许临把手机扣过去。

他点开 `03_颁奖词_云底`。

舞台灯光很亮。孟青桐拿着奖杯，身边站着几个嘉宾。

孟青桐：谢谢大家，也谢谢云帝哥。

台下有人喊：

观众：是云底吧！

台上几个人笑。

孟青桐握着话筒，看向李云帝那边。

李云帝也笑了一下。

孟青桐：不是那个帝，是底下托着我们的底。

台下掌声和笑声混在一起。

李云帝脸上的笑慢慢收掉。

孟青桐继续说：

孟青桐：因为他一直在后面托着大家。

许临按空格。

画面停在“底下托着我们的底”。

他把后半句拖到素材箱底部。

改名：

`暂不用_解释`

甲方语音弹出。

甲方语音：这个就是爆点。别解释，一解释就软。

许临把“云底”那一句复制到时间线。

字幕：

`当众改名`

他又加一行：

`只是玩笑吗？`

问号停了两秒。

他删掉问号。

`只是玩笑。`

许临盯着这四个字。

删掉。

改成：

`不是玩笑。`

雨水从窗台滴到地板上。

咚。

他点开 `04_不接话`。

素材里，李云帝把话筒往孟青桐那边递。

李云帝：你当年第一部戏不是我——

孟青桐低头看手机。

李云帝停住。

另一位嘉宾接话。

屏幕上是节目组工作群：

`下一轮流程改了，青桐看手机确认。`

只剩孟青桐低头。

李云帝停住。

另一位嘉宾接话。

许临加红圈，圈住孟青桐的眼睛。

红圈跟不上她低头的动作，抖了一下。

他重新打关键帧。

红圈稳住。

门外有脚步声经过。

手机视频声：男明星一句不舒服，到底忍了多久？

竖屏直播。孟青桐坐在化妆间，妆发做到一半。

孟青桐皱了一下眉。

孟青桐：小孩真的好吵。

孟青桐看了一眼镜头。

孟青桐：我知道。我就是听不了这种尖的。

许临把这段拖进时间线。

评论截图随素材一起弹出：

`她以前就这样，没同理心`  
`怪不得会霸凌人`  
`这姐是真冷`

备注：

`尾款，成片后结`

许临把截图拖到桌面角落。

桌面角落还有上一版视频的收益预估。

两个小窗口叠在一起。

这一次，他没有立刻调整音量。

## 编剧自检

- 第二次站队加强：孟青桐现在有几处真实不体面/不周到行为，观众可以合理地觉得她也伤害过人。
- 仍未把她写成绝对反派：保留“别这么说”、工作群、后半句解释、道歉声明等复杂上下文，但不提前喂给观众。
- 核心主题仍是剪辑机制：同样通过裁剪、放大、静音、标题和评论截图，把“不体面”加工成“有罪”。

## 留给 Human 的不确定点

- “小孩真的好吵”是否太容易把女明星推成道德靶子，可后续用更具体上下文回收。
- “云底”现场笑是否足够刺人，还是需要加一个李云帝当场短暂停顿的近景。
- 这版是否已经让你感觉“女的也确实有点问题”，但还没有把电影变成审判她。
