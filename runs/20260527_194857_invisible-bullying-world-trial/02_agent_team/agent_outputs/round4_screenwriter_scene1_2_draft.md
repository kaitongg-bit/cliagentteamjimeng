# 第 1-2 场正文初稿

【场景1：内景 许临出租屋 / 夜】

黑屏。

一声鼠标点击。

剪辑软件亮起。时间线密密麻麻，项目名停在左上角：

`male_guilty_final_02`

播放窗口里，李云帝坐在综艺圆桌正中。身边几个人笑着接话。画面定格在他低头的一帧。

手机震动。

屏幕右上角弹出房东消息：  
`明早十点前。别让我再催。`

许临坐在电脑前，三十二岁，胡茬没刮，外卖袋压着半张欠费单。他把消息划掉，指尖停在空格键上。

又一条消息进来。甲方头像没有名字，只有一个灰色字母。

甲方语音：许老师，急活。今晚定。明早八点前要上。

许临点开尾款截图。数字不大，但够房租。

他把耳机戴上，拉近时间线。

许临：方向？

甲方语音：别中立。现在中立没人看。

许临新建一条字幕轨。光标在标题框里闪。

甲方语音：观众点进来，三秒就得知道该骂谁。

许临删掉标题框里的空格。

许临：素材全吗？

甲方发来一个压缩包。

文件名：  
`李云帝风波_可用素材_pack`

许临下载。进度条跑得很慢。房东消息又弹出来，盖住进度条一角。

甲方语音：先走男的。强者卖惨，粉丝下场，女方被网暴。你干过综艺，知道怎么剪。

许临没回。他把房东消息拖到屏幕边缘。

压缩包解开。

文件夹一排排展开：

`01_主位`  
`02_他说不舒服`  
`03_众人陪笑`  
`04_粉丝控评`  
`05_旧综艺绊倒`  
`06_原始长镜头`

鼠标停在 `06_原始长镜头` 上。

停了两秒。

甲方又发来一条文字：  
`标题先定，别被素材带跑。`

许临点开 `01_主位`。

监看窗口里，李云帝坐在桌子正中，灯光打得最亮。孟青桐在他左侧，低头笑了一下。

许临把这一段拖进时间线。

他点开标题框，输入：

`他一句“不舒服”，毁了别人半生？`

标题盖住李云帝的脸。

手机屏幕暗下去。房间里只剩电脑风扇声和素材里的掌声。

许临按下保存。

项目名变成：

`male_guilty_final_03`

切。

【场景2：内景 许临出租屋 / 电脑时间线 / 夜】

时间线放大到帧。

第一段：李云帝坐主位。  
第二段：众人笑。  
第三段：李云帝对镜头说“我确实有点不舒服”。

许临把第三段往前拖了半秒。

李云帝：我确实有点不舒服。

下一帧立刻接上孟青桐的笑。

许临回放。

李云帝：我确实有点不舒服。

孟青桐笑声响起。

许临按暂停。切点落在孟青桐抬眼前一帧。

他把笑声复制，贴到前面一点。再回放。

笑声提前了。

李云帝低头的四秒变长。

许临打开音乐库，试听几条音轨。

轻快鼓点。

他删掉。

低频鼓声。

他拖进音轨，压在李云帝低头那四秒下面。

监看窗口里，李云帝的沉默变沉。字幕弹出：

`他早就不满？`

许临把问号删掉。

字幕变成：

`他早就不满。`

微信弹出甲方文字：  
`别问句，问句软。`

许临没回。他点开 `05_旧综艺绊倒`。

旧综艺画质更糊。李云帝在竞技游戏里伸手，旁边男嘉宾摔倒。现场笑成一片。

许临把时间戳裁掉，调成和前面一样的冷色。

屏幕上出现新标题：

`不是第一次了`

他把旧综艺片段接在“不舒服”后面。

又点开 `04_粉丝控评`。

一张张截图快速铺进时间线：

`哥哥只是说实话`  
`孟青桐出来道歉`  
`别装无辜`  
`品牌还不撤？`

许临把截图速度调快。评论像雨点一样闪过。

他又拖进一张热搜截图：

`李云帝 不舒服`

热搜后面跟着一个红色“爆”。

许临放大红字，裁掉截图右下角日期。

回放。

李云帝坐主位。  
众人笑。  
李云帝低头。  
低频鼓声。  
“不舒服”。  
旧综艺摔倒。  
粉丝截图。  
热搜爆字。

许临摘下一边耳机。素材原声从另一边漏出来。

原始素材里，有人喊：

节目工作人员：云帝哥，您坐中间，我们机位好架。

许临按空格。

画面停在工作人员伸手指座位的一帧。

他把这一小段拖到素材箱底部，改名：

`暂不用`

甲方语音弹出。

甲方语音：有旧料吗？别只剪这次，显得小题大做。

许临看着 `暂不用` 的缩略图。

他点开旧料文件夹，把摔倒片段又复制了一遍。

标题轨上，新的大字压下来：

`他把“不舒服”变成武器`

许临拖动音量线，把李云帝原声压低，只留下“我不舒服”三个字更清楚。

李云帝：我……不舒服。

孟青桐的笑声接上。

许临把笑声加了混响。

房间门外，有邻居拖行李箱经过。轮子声短短擦过地面。

许临没有抬头。

导出窗口弹出。

文件名自动填入：

`male_guilty_final_03_REALFINAL.mp4`

他把鼠标移到“导出”。

停住。

时间线底部，`暂不用` 的缩略图还在。李云帝坐在桌边，工作人员的手停在半空。

许临把导出窗口挪到一边，露出缩略图。

甲方文字跳出：  
`先发，热度窗口快过了。`

许临点下“导出”。

进度条从 1% 跑起。

低频鼓声还在耳机里循环。  
李云帝的声音被切成很短的一句：

李云帝：不舒服。

进度条到 100%。

# 编剧自检

- 未写心理描写、括号情绪提示、解释性主题对白。
- 戏剧动作集中在剪辑动作：拖拽、裁切、静音、加标题、换配乐、导出。
- 第 1-2 场让观众先接受“男明星强者卖惨/操控舆论”的框架，但用 `暂不用` 和座位安排机位埋下未定性空间。
- 没把李云帝或孟青桐定成最终罪人，罪感来自剪辑顺序、标题和声音。

# 留给 Human 的不确定点

- 第 1 场是否要更早露出孟青桐，还是继续把她压到第 2 场末端。
- “毁了别人半生”是否过猛，可改成更平台化的标题。
- 许临是否需要一句更生活化的自嘲台词，强化失业外包处境。
