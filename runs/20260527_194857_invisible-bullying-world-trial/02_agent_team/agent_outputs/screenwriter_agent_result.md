# Screenwriter Agent Result

已按 canonical skill 路由到 **5-10 分钟叙事短片**，本轮只做“破题与核心动作 + 3 个故事方向”。

## 方向 A：《未发送》

**Logline：** 一个靠剪辑热点糊口的失业青年，接到一单“把男明星不舒服做成全网玩梗模板”的活，直到他发现自己十年前被同学排挤的旧视频，也被网友当成同款素材转发。

**核心戏剧动作：**  
主角要在今晚 12 点前交出一条爆款二创视频，换来急需的房租钱；阻碍是素材里那些“苦笑、对视发笑、被排除在群外”的细节，越来越像他自己的旧伤。

**主角 Want / Need：**  
Want：做出爆款，拿到结算，证明自己还能在这个城市活下去。  
Need：承认“我只是转发一下”也是动作，不是无害的旁观。

**核心阻碍：**  
平台算法、甲方催稿、评论区的即时奖励、经济压力，让他每一次犹豫都显得“矫情”；同时，他的旧同学在群里也开始转发那个明星梗。

**结尾画面：**  
凌晨，主角把剪好的视频拖进回收站。电脑屏幕上，另一个账号已经发出同款视频，点赞数跳得飞快。他关掉电脑，手机震动不停，黑屏映出他的脸。

**如何转向审判网友和时代情绪：**  
男女明星只作为屏幕里的素材存在，不给他们定罪。镜头真正跟随一个普通网友：他不是恶人，只是缺钱、焦虑、想被看见、想跟上热闹。审判落在“低成本发泄如何变成产业链”：甲方、热搜、平台反馈、群聊嘲笑，共同把一个人的“不舒服”加工成商品。

## 方向 B：《包场群》

**Logline：** 一个综艺后期剪辑师发现，自己剪出的“朋友们给明星惊喜”的温情段落，在播出后被网友逐帧拆成霸凌证据；当舆论反转，他又被要求剪出相反版本。

**核心戏剧动作：**  
主角要在一夜内交出节目收官彩蛋，把一场尴尬饭局剪成“朋友团圆”；阻碍是同一批素材无论怎么剪，都能被剪成两种完全相反的故事。

**主角 Want / Need：**  
Want：按时交片，保住工作，别让项目组背锅。  
Need：看见“剪辑”和“转发”一样，都在参与制造现实，而不是单纯记录现实。

**核心阻碍：**  
节目组要温情，艺人团队要体面，营销号要冲突，网友要站队。主角手里只有同一组画面：一个苦笑、一次对视、一个没拉人的群、几句玩笑话。

**结尾画面：**  
主角把时间线上的两版视频并排播放：左边是“他被霸凌”，右边是“她被网暴”。两个画面在同一个苦笑处暂停。主角摘下耳机，办公室里全是机器风扇声。

**如何转向审判网友和时代情绪：**  
它不审判男明星或女明星，而是审判“叙事机器”：同一事实被不同需求剪成不同武器。网友不再只是评论者，而是二次剪辑、二次定罪、二次消费的人。时代情绪体现在所有人都急着从混乱里捞到一点确定感：节目要热度，打工人要饭碗，网友要出口。

## 方向 C：《一星》

**Logline：** 一个外卖骑手为了拿平台奖励，跟风给男明星父亲的餐馆刷一星差评；送餐途中，他遇到一个被同事排挤的女孩，发现自己正在把别人的伤口当成今日任务。

**核心戏剧动作：**  
主角要完成“热门商家差评截图”的兼职任务，换一笔小钱；阻碍是他当天不断撞见现实里的隐形排挤：没人等电梯、群里不回消息、饭桌上突然发笑。

**主角 Want / Need：**  
Want：多赚两百块，补上电瓶车维修费。  
Need：停止把自己的憋屈转嫁给一个没见过的人。

**核心阻碍：**  
他穷、累、被平台罚款、被顾客催、被生活逼到没有余裕；群聊里所有人都在刷“不舒服文学”，他不跟就显得落伍。

**结尾画面：**  
主角站在餐馆门口，手机停在一星评价页面。他抬头看见店里一个中年男人把差评单一张张撕下来，又重新贴上“今日正常营业”。主角删掉评价，把外卖箱扶正，骑进早高峰。

**如何转向审判网友和时代情绪：**  
明星事件只是一条热搜，主角是完全陌生的普通人。故事把“全网玩梗”落到现实伤害：一星差评、陌生人的生意、被波及的家人。审判对象不是某个粉圈，而是经济下行里人人憋着一口气，最后把最便宜的攻击投向最远的人。

## 推荐方向：A《未发送》

最适合 10 分钟。

原因是它的空间最集中：一间出租屋、一台电脑、几个群聊、几段素材，就能完成完整戏剧动作。主角的外部目标很清楚：今晚交爆款拿钱；内部变化也能在 10 分钟内完成：从“我只是做个梗”到“我也在动手”。它还能最直接实现用户核心目标，让观众意识到：伤害不只发生在明星之间，而发生在每一次剪辑、转发、跟笑、玩梗里。
