# Audience Agent Result

## 1. 三类观众反馈

### A. 电影节观众

**最想看的东西**

- 男明星“不舒服”的复杂性：他既有权力、资源、体面，也真的受伤。
- 女明星不是反派，而是一个在糊、焦虑、讨好小团体、怕失去机会的人。
- 网友不是抽象恶魔，而是一个个生活困顿、情绪无处安放的人。
- 舆论如何从“看热闹”变成“集体审判”，再变成无人负责的伤害机器。
- 结尾有余味：不是谁赢了，而是所有人都被这个时代污染了。

**最可能反感的东西**

- 把网友拍成低智反派。
- 用台词直接喊主题，比如“你们才是真正的凶手”。
- 现实事件复刻感太强，像八卦复盘，不像微电影。
- 男明星被拍得太圣人，女明星被拍得太坏。
- 最后靠官媒、热搜反转完成“正义归位”，会显得简单。

**最容易看不懂的地方**

- “隐形霸凌”如果只靠建群、对视笑，可能不够有电影性的累积。
- “审判网友”如果没有具体网友角色承载，会显得主题漂浮。
- 经济下行与娱乐舆论之间的关系，如果只点一句背景，会显得硬贴议题。

### B. 普通短视频观众

**最想看的东西**

- 开场就有强冲突：大明星在自己的节目里被排除、被笑、被孤立。
- 情绪明确：他为什么“不舒服”，观众要立刻感受到。
- 反转信息：原来那个群不是单纯排挤？原来女明星也有委屈？原来网友剪辑断章取义？
- 爽点边缘但不彻底反杀：观众会想看“到底谁错了”，但最后要让他们发现“我也可能错了”。
- 具象伤害：爸爸餐馆被差评、普通人跟风玩梗、当事人沉默崩塌。

**最可能反感的东西**

- 节奏太慢，前 2 分钟都在铺节目关系。
- 人物说话太文艺，像论文。
- “网友有罪”讲得太直，会让观众防御：我只是看个热闹，凭什么骂我？
- 结尾太灰，完全没有情绪释放。
- 信息太多：建群、颁奖词、剪辑、营销号、官媒、粉丝、厌童言论，全塞进 10 分钟会乱。

**最容易看不懂的地方**

- “你只是个李云底/拖底”这种谐音梗，如果没有视觉化解释，会卡住。
- 谁是朋友、谁是老板、谁是被托举者，关系需要极快建立。
- 舆论反转不能太依赖微博机制，否则非饭圈观众会只觉得“明星又吵架”。

### C. AI 视频创作者观众

**最想看的东西**

- 可生成、可传播的强画面：弹幕洪水、热搜审判庭、餐馆差评雨、手机屏幕映在人脸上。
- 明确的段落结构，方便做成 10 分钟短片：节目现场、热搜发酵、全民玩梗、现实伤害、审判反照。
- 高概念设定：网友评论像法庭证词，算法像无形法官。
- 角色脸谱不能太复杂，但情绪轨迹要清楚。
- 结尾能出一句短视频传播级记忆点，但不能像口号。

**最可能反感的东西**

- 全靠聊天截图和微博界面，视觉单调。
- 主题很大但没有能落地生成的场景。
- 人物过多，AI 视频难以保持一致性。
- 现实娱乐圈指涉太强，容易变成仿真人物争议。
- 画面先行压过故事，比如一上来就是宏大赛博审判，但观众还不知道为什么审判。

**最容易看不懂的地方**

- 谁在被审判：男明星？女明星？网友？世界？
- “时代情绪”如果没有具象角色，很难生成成画面。
- 反转如果靠大量文字信息，AI 视频观众会觉得不适合影像化。

## 2. 主题如何说服观众，而不是说教？

观众需要看到三件事。

**第一，网友不是天生恶人，而是一步步滑进去的。**  
比如一个失业青年、一个宝妈、一个加班到深夜的白领，他们本来只是刷到梗，笑了一下，转了一下，评论了一句“不舒服文学又来了”。他们没有恶意计划，但每个人都贡献了一点重量。

**第二，伤害必须落到现实身体上。**  
不是“男明星心情不好”这么轻，而是：他爸爸餐馆被陌生人打一星；员工接电话被骂；父亲低头擦掉门口被贴的梗图；男主角看到父亲说“没事，生意不好本来也正常”。这比喊“网络暴力害人”更有力。

**第三，要让观众发现自己曾经站在加害位置上。**  
关键不是让角色控诉网友，而是设计一个瞬间：观众前面也觉得好笑的梗，在后面变成伤害证据。比如前半段观众笑“不舒服文学”，后半段同一句话出现在餐馆差评、遗照式剪辑、陌生人私信里，笑点变成刺点。

最有效的表达方式：  
不要说“你们都是凶手”，而是让观众意识到：“我刚才笑的那个东西，原来落到别人身上是这样的。”

## 3. 10 分钟微电影最有记忆点的 5 种可能画面

### 1. 自己的节目，没有自己的群

男主角站在后台，其他 5 个人手机同时震动、低头笑。他问“什么事？”所有人瞬间收手机，说“没事”。镜头停在他脸上的苦笑。

### 2. 热搜变成审判席

手机屏幕上的评论一条条弹出，逐渐变成法庭旁听席。网友头像坐满高处，敲键盘声像法槌声。男主角和女主角都站在被告席，但真正的法官座位是空的。

### 3. “不舒服文学”满城复制

电梯广告、奶茶杯贴纸、公司群表情包、餐馆差评、直播弹幕，全都出现同一句“不舒服”。它从玩笑变成城市噪音。

### 4. 父亲餐馆的一星雨

餐馆墙上挂着男主角小时候照片。父亲正在给客人端菜，手机不断响起差评提醒。门外有人拍视频说：“这家就是那个不舒服他爸开的。”父亲把“欢迎光临”的牌子翻成“暂停营业”。

### 5. 网友看见自己的评论

结尾，一个普通网友走进类似审判庭的空间，发现大屏上不是明星黑料，而是自己随手发过的评论。她想解释“我只是跟风”，但周围所有人也都在说同一句：“我只是跟风。”

## 4. 最可能让观众想继续看的开场钩子 3 个

### 钩子 1：审判庭开场

黑屏，键盘声像雨。  
男主角站在被告席，女主角也站在被告席。旁听席全是发光头像。

法官席空着。

字幕/旁白：  
“今天审判的，不是他们。”

然后切回节目后台：其他人手机同时震动，只有他没有。

### 钩子 2：一句“不舒服”引爆世界

男主角深夜发微博：  
“我真的有点不舒服。”

下一秒，手机像炸弹一样震动。  
热搜、鬼畜剪辑、表情包、餐馆差评、营销号标题快速闪过。

最后切到父亲餐馆电话响起。  
电话那头说：“你儿子这么玻璃心，你家菜是不是也不舒服？”

### 钩子 3：他以为这是为朋友开的节目

节目开录前，男主角给每个人准备礼物，认真记他们的咖啡口味、镜头角度、出场机会。

导演问：“你确定要把这么多镜头分给他们？”

他说：“他们只是缺一次被看见。”

下一场，所有人围在一起笑。  
他走近，笑声停了。  
他问：“你们刚才在聊什么？”  
没人回答。
