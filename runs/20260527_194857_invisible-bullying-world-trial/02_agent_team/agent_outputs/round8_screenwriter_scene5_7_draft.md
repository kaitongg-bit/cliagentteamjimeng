# 第 5-7 场正文初稿

【场景5：内景 许临出租屋 与 楼道 / 凌晨】

导出条走到尽头。

文件名停在桌面：

`female_bully_final_02_REALFINAL.mp4`

许临的手指悬在触控板上。

手机震动。

房东消息：

`十点前。`

甲方文字：

`发。`

许临把视频拖进发布后台。

封面自动生成：孟青桐低头笑，李云帝抬眼看她。

标题栏里是他刚才打好的两行字：

`孟青桐道歉前，没人敢放这几秒`

`她们的玩笑，为什么只有他笑不出来？`

许临删掉第二行的问号。

发布。

后台弹出：

`审核通过`

`发布成功`

评论开始刷新。

`终于有人说了`  
`她不是第一次这样`  
`别装无辜`  
`品牌还不撤？`

许临把上一条视频后台点开。

两个窗口并排。

左边是李云帝那条。

`终于有人说了`  
`他不是第一次这样`  
`别装无辜`  
`品牌还不撤？`

右边是孟青桐这条。

`终于有人说了`  
`她不是第一次这样`  
`别装无辜`  
`品牌还不撤？`

许临拖动鼠标，把两边评论时间戳对齐。

几乎重合。

门外传来电梯到达声。

楼道里，外卖员一手拎着两份餐，一手刷手机。手机里正播孟青桐那条视频。

视频声从门缝挤进来。

孟青桐：那今天就叫云底哥吧。

低频鼓声压上去。

外卖员停在隔壁门口，敲门。

外卖员：外卖。

隔壁没人应。

他低头继续看，评论区盖住画面。

外卖员：这女的挺会的。

许临坐在门内，没动。

隔壁门开了半条缝。一个邻居伸手接餐。

邻居：又谁啊？

外卖员：孟青桐。热搜第一。

邻居：前两天不是那个男的？

外卖员：都一样。

他把餐递过去，空出拇指，点了个赞。

外卖员：给个好评啊。

电梯门合上。

许临打开门。

楼道灯坏了一盏，剩下那盏一闪一闪。电梯门上贴着一张外卖平台广告，代言人是孟青桐。有人用黑笔在她脸旁写了两个字：

`道歉`

字还没干，被雨衣袖子蹭花了一道。

许临站了一会儿，伸手按电梯。

电梯停在一楼，没有上来。

手机震动。

甲方语音：起来了，继续盯。她那边肯定要回应，回应前我们再补一条。

许临回到屋里。

桌上的外卖汤已经凉透，油凝成一层白边。窗台的水顺着墙线往下流，滴到插线板旁边。

他把插线板往外拖。

电脑提示音连续响。

后台曲线往上抬。

右边窗口刷新出新评论。

`她妈开的那个花店是不是也叫青桐？`  
`别光骂，搜得到地址`  
`一家人吃红利吧`

许临把窗口关掉。

评论声音还在后台自动播放。

孟青桐：不是那个帝，是底下托着我们的底。

笑声被重复三遍。

切。

【场景6：内景 许临出租屋 / 凌晨将尽】

雨小了一点。

屋里没有开灯。电脑屏幕上，素材库多了一个新文件夹：

`07_多机位原始_未剪`

文件夹旁边还有一条甲方文字：

`有人放全机位了。别管。我们先出下一条。`

许临点开。

第一段是圆桌全景。

主持人：云帝，你刚才说不舒服，是身体不舒服，还是这个环节不舒服？

李云帝拿着水杯，杯口碰到嘴边，又放下。

李云帝：都有点吧。录太久了。

主持人笑。

主持人：那青桐说两句？你们认识最久。

孟青桐低头看手机。

手机屏幕上，工作群消息一条条跳：

`下一轮改流程`

`青桐看一下台本`

`不要让云帝接游戏惩罚`

李云帝看她一眼。

另一位嘉宾接话。

全景里，孟青桐抬头慢了半拍。她把手机扣在腿上。

孟青桐：他今天确实累了。

主持人已经转向别人。

许临把进度条往后拖。

颁奖现场全机位。

孟青桐：那今天就叫云底哥吧。不是那个帝，是底下托着我们的底。

台下笑。

李云帝跟着笑了一下。

孟青桐继续说：

孟青桐：因为他一直在后面托着大家。

掌声起来。

另一台机位里，李云帝低头整理袖口，笑慢慢收掉。

许临停住。

他点开群聊完整截图。

有人发：

`生日局别拉他吧，他来了大家都要围着他转。`

有人接：

`他会不会又说自己不舒服哈哈。`

孟青桐隔了十几秒才回：

`别在大群说。`

再隔几秒。

`先小群定吧，省得他又不舒服。`

截图往下，还有一行。

`回头单独约他吧，今天人多。`

许临把鼠标移到这一行，又移开。

他打开 `道歉声明草稿`。

几行字停在屏幕上：

`我没有恶意。`

`但我的表达确实让人不舒服。`

`我应该更早意识到。`

许临把这几行拖进时间线。

又删掉。

他打开李云帝那条的原始长镜头。

李云帝坐在圆桌主位，摄像机重新调焦。

李云帝：我不想玩那个互相整人的环节。别拍成我扫兴。

工作人员：明白，明白。

旁边有人递来台本。

李云帝翻了两页。

李云帝：孟青桐那段别剪太满，她容易被骂。

工作人员：这个看后期。

李云帝没再说。

许临又点开另一段后台花絮。

孟青桐站在舞台侧面，拿着奖杯。

孟青桐：我刚才那个“云底”是不是有点过？

助理：没事，大家都笑了。

孟青桐看向台口。

李云帝正在跟品牌方合影，脸上挂着笑。

孟青桐：那就算了。

助理：走了，车到了。

孟青桐把奖杯递给助理，跟着走。

许临关掉素材。

房间里只剩雨后的滴水声。

甲方语音弹出。

甲方语音：下一条就打“道歉声明漏洞”。她说没有恶意，你就问没有恶意就不用负责吗。

许临没有点开语音条。

他在聊天框里输入：

`全素材不够定她。`

停了一下。

又补：

`也不够定他。`

发送。

很快，对面回复。

甲方文字：

`我们不是法院。`

第二条紧接着来：

`尾款你还要不要？`

许临看着那两行字。

桌边的手机亮起，银行卡到账提醒弹出一半，又被低电量提示盖住。

许临把电源线插上。

插头旁边，水珠沿着桌腿落到地上。

他弯腰，把插线板踢远一点。

屏幕上，甲方又发：

`确定一点。大家就买这个。`

许临没有回。

他新建项目。

项目名：

`not_enough_to_convict`

光标闪了几下。

他把两个原始长镜头拖进去。

时间线空了很久，才出现第一段完整声音。

没有鼓声。

没有红圈。

没有字幕。

# 编剧自检

- 完成第二条反噬：同一外卖员、同一评论句式、同一“确定性消费”换了主语。
- 没有洗白双方：李云帝、孟青桐都有自保、虚荣、误伤和回避，但都不足以被定罪。
- 完成最终结构：许临停手无效，系统继续复制、发布，并让“剪辑师本身”也进入被剪辑的审判。

# 留给 Human 的不确定点

- 第 7 场“剪辑师被剪”是否需要再更冷一点，减少陌生视频的直白字幕。
- 第 6 场完整素材的信息量是否略密，可在定稿时压缩成两段核心长镜头。
- 外卖员“都一样”这句是否保留，或改成更生活化的无意识重复。
