# 第 3-4 场正文初稿

【场景3：内景 许临出租屋 / 发布后台 与 城市雨夜 / 夜】

导出完成提示音。

文件停在桌面：

`male_guilty_final_03_REALFINAL.mp4`

许临把视频拖进发布后台。

封面自动生成：李云帝低头，孟青桐的笑停在他身后。

标题栏里已有一行字：

`他把“不舒服”变成武器`

许临看着封面。

窗台下的旧台本被水泡软，纸面塌下去一点。水没有再顺进盆里，沿着墙往下流。

手机震动。

房东消息：  
`十点。`

甲方文字：  
`发。`

许临点下发布。

后台弹出：

`审核通过`

`发布成功`

房间里很静。只有雨声和电脑风扇声。

几秒后，评论开始刷新。

`终于有人说了`  
`强者卖惨最恶心`  
`孟青桐被他粉丝骂成那样，他还装？`  
`他爸不是开店吗`

许临把后台窗口缩小，点开收益预估。数字跳了一下，很小。

他拿起桌边的冷馒头，咬了一口。

视频声音从电脑里自动播放。

李云帝：我……不舒服。

低频鼓声接上。

切到——

【外景 小区门口 / 夜】

雨棚下，外卖员把手机夹在电动车把手上，等红灯。

同一条视频在竖屏里播放。标题挤在李云帝脸上。

外卖员拇指一划，评论区盖住画面。

他笑了一下，把手机音量调大。

旁边一个穿雨衣的女人看了一眼屏幕。

女人：又谁塌了？

外卖员：不知道。热搜第一。

绿灯亮。

电动车冲进雨里。手机里还在响。

李云帝：不舒服。

切回——

【内景 许临出租屋 / 夜】

后台曲线往上抬。

评论刷新更快。

`李记汤面是不是他爸开的？`  
`去打分啊，别光骂`  
`一家人都吃红利吧`

许临的鼠标停住。

他点开一张评论截图。截图里是一家小餐馆的页面，店名：李记汤面。评分旁边的星星正在变少。

新差评一条条冒出来。

`儿子这样，家里也不会好`  
`不吃了`  
`退单`

许临把截图关掉。

门铃响。

他起身开门。

门外是外卖员，雨水顺着头盔边往下滴。外卖袋上贴着促销券。

外卖员把袋子递给他，手机还亮着，正播许临刚发的视频。

视频里，孟青桐笑声被混响拉长。

外卖员扫了一眼许临屋里的电脑。

外卖员：你也看这个啊。

许临接过袋子。

许临：嗯。

外卖员：这男的挺会的。

许临没接话。

外卖员看手机。

外卖员：哎，评论说他爸店被冲了。

楼道灯灭。

外卖员按了一下墙上的开关，灯又亮。许临的脸被白光照了一下。

外卖员：给个好评啊。

许临点头。

关门。

外卖袋放到桌上，袋底蹭过一张欠费单。汤洒出一点，油顺着塑料盒边缘流到桌面。

电脑提示音连续响。

甲方发来语音。

甲方语音：可以。先别停。

许临坐回去。

甲方又发来一条。

甲方语音：评论区开始转了。有人扒孟青桐那边。你接着做。

许临看着发布后台。视频封面上的李云帝被弹幕一层层盖住。

甲方发来新压缩包。

文件名：

`孟青桐_隐形霸凌素材_pack`

下面跟一条文字：

`现在大家想看这个。`

许临没有动。

窗台上的旧台本彻底塌下来，水直接滴到地板上。

咚。

咚。

咚。

甲方文字：  
`热度窗口还在。`

许临拖过纸巾，擦掉桌上的汤油。纸巾越擦越花。

他点下下载。

进度条开始跑。

【场景4：内景 许临出租屋 / 素材库新文件夹 / 夜】

压缩包解开。

文件夹铺满屏幕：

`01_群聊不带他`  
`02_对视笑`  
`03_故意不接话`  
`04_颁奖词漏人`  
`05_避开同框`  
`06_多机位原始`

许临新建项目。

项目名闪了一下：

`female_bully_final_01`

他点开 `01_群聊不带他`。

一张聊天截图打开。群名被红框圈住：  
`青桐生日局`

截图里没有李云帝。

许临把图片拖进时间线，放大群名，裁掉上方日期和下方几条消息。

裁掉前，最底下一行露出半句：

`云帝那天录制到凌晨，别叫他了`

许临的鼠标停在裁剪框边缘。

甲方文字弹出：  
`群聊能用，别放太全。`

许临把裁剪框往上收。

那半句消失。

他点开 `02_对视笑`。

监看窗口里，孟青桐和另一位女嘉宾对视，笑了一下。

许临把这段接到李云帝低头后面。

回放。

李云帝低头。  
孟青桐笑。  
另一位女嘉宾低头捂嘴。  

许临复制上一场用过的笑声波形，贴到这里。

笑声提前两秒。

他加字幕：

`所有人都懂，只有他坐在那里。`

他看了一眼字幕，删掉“所有人”。

改成：

`她们都懂。`

手机震动。

房东消息：  
`收到钱我就不说了。`

许临把手机扣过去。

电脑里，素材继续播放。

节目工作人员：青桐，接一下，别冷场。

许临按空格。

画面停住。

他点开 `06_多机位原始`。

屏幕变成四宫格。

A机位：孟青桐笑。  
B机位：李云帝低头翻台本。  
C机位：工作人员举着手卡，手卡上写着：`笑一下，转场`。  
D机位：另一位嘉宾低声对孟青桐说话。

另一位嘉宾：导演让我们带一下。

孟青桐：嗯。

她拿起话筒，往李云帝那边递了一点。

孟青桐：云帝哥，你刚不是说——

李云帝摆了摆手，没接。

画面里，导演的声音压过来。

节目工作人员：好，过，下一趴。

许临回放这一小段。

又回放一次。

甲方语音弹出。

甲方语音：许老师，别做纪录片啊。我们要短视频。

许临关掉四宫格。

只留下A机位里孟青桐笑的特写。

其他机位变灰，静音。

他点开 `04_颁奖词漏人`。

台上灯光很亮。孟青桐拿着奖杯，旁边站着几位嘉宾。

孟青桐：谢谢我们这桌所有女生，真的，谢谢你们一直托着我。

许临切在这里。

下一秒，未剪完的原素材继续往下放。

孟青桐：还有云帝哥，刚才一直帮我们补话，辛苦了。

现场掌声响起。

许临把播放头拖回去。

再次切在“托着我”后面。

掌声接上。

字幕：

`只谢女生，不提同桌男嘉宾`

他把“男嘉宾”改成“他”。

`只谢女生，不提他`

雨声忽然变大。

窗外一道车灯扫过，屏幕反光里，许临的脸和孟青桐的脸重在一起。

他点开 `05_避开同框`。

综艺后台，孟青桐从李云帝身边走开。

许临拖进时间线。

原素材里，画面边缘有工作人员伸手指位置。

节目工作人员：青桐姐，站左边，logo别挡。

许临把画面放大。

工作人员的手被裁掉。

孟青桐只剩一个转身离开的背影。

他调慢到 0.75 倍。

转身变长。

李云帝站在原地，看向另一边。

许临把上一场那句李云帝的声音拖过来，压在画面下。

李云帝：不舒服。

这一次，低频鼓声没有进来。

只有雨声。

许临盯着音轨，把低频鼓声重新拖进去。

房间门外，邻居经过，手机外放传进来。

手机视频声：他一句“不舒服”，到底忍了多久？

脚步声远去。

许临打开标题框。

输入：

`这不是玩笑，是围猎`

他停了停。

删掉。

输入：

`孟青桐道歉前，没人敢放这几秒`

又删掉。

甲方文字：  
`标题别怂。`

许临点开 `03_故意不接话`。

素材里，李云帝说话。孟青桐低头看手机，没有接。

许临把它拖进时间线。

原素材往前多出三秒。

孟青桐的手机屏幕亮着，是节目组工作群：

`下一轮流程改了，青桐看手机确认`

许临把入点往后拖。

手机屏幕消失。

只剩孟青桐低头不接话。

他加上红圈，圈住她的眼睛。

播放。

李云帝：那我就先——

孟青桐低头。

李云帝停住。

孟青桐的笑声接上。

标题框再次亮起。

许临输入：

`她们的玩笑，为什么只有他笑不出来？`

他没有删。

甲方发来一个转账预览截图。

备注：  
`尾款，成片后结`

许临看着截图。

桌上的外卖汤已经凉了，油凝在盒盖边缘。

他按下保存。

项目名变成：

`female_bully_final_01`

时间线里，李云帝的低头、孟青桐的笑、群聊截图、颁奖词断句、转身背影连在一起。

许临回放。

孟青桐：谢谢我们这桌所有女生，真的——

切断。

李云帝：不舒服。

窗台的水滴落进空塑料盆。
