# Screenwriter Agent Feedback: Editor Reveal

已按 `/Users/bytedance/Downloads/screenwriting-master.skill` 路由到 **5-10 分钟叙事短片**。结论：这个新结构适合 10 分钟微电影，但必须把“剪辑软件时间线揭示”处理成**伦理反转**，不是**真假反转**。

## 1. 最大优点

它让观众亲身完成一次“被剪辑带着站队”的体验。

原方案《未发送》的锋利点是：主角参与制造舆论伤害。新结构更进一步：连观众也被纳入制造链。前 8-9 分钟不是单纯讲“网友容易被带节奏”，而是让观众自己先被叙事、镜头、素材顺序、配乐、留白牵引，产生判断。结尾退到时间线后，观众意识到：刚才我以为自己在判断真相，其实我也在消费一个被排列过的版本。

这非常贴合原始目标：审判的不是男明星或女明星，而是“确定性饥渴”、舆论机制、观看欲望本身。

## 2. 最大风险

最大风险是变成 cheap twist：最后告诉观众“刚才都是剪出来的”，等于把前面 8-9 分钟的情感和伤害取消掉。

必须避免这几个误区：

- 不要让观众觉得“所以男明星没被伤害，女明星也没问题，一切都是假素材”。
- 不要把结尾拍成炫技式反转：哇，原来是剪辑软件。
- 不要让剪辑师像幕后黑手、操控大师，那会把系统性问题缩小成一个坏人。
- 不要让结尾只是在说“影视都是假的”，那会从社会议题滑向元电影小聪明。

正确方向是：**素材可能都是真的，但观众的确定性是被制造的。**

也就是说，结尾不是推翻前文，而是污染前文。观众仍然相信有人受伤、有人被网暴、有人把痛苦变成笑料；但他们不再敢相信自己刚刚站队时的干净感。

## 3. 推荐改写 Logline 和核心戏剧动作

推荐新版 logline：

> 一个深夜赶工的后期剪辑师，必须在午夜前剪出一条足够“有立场”的舆论短片：让观众相信一位男明星被朋友隐形霸凌，也相信嘲笑他的人都该被审判。随着素材越剪越顺，他终于完成了一版锋利、煽动、近乎正义的成片。可最后，画面退回剪辑软件时间线：刚才所有让观众愤怒、同情、站队的顺序、停顿和沉默，都是他亲手摆放的。

核心戏剧动作建议从原来的：

> 主角必须在午夜前交出一条嘲讽男明星的爆款视频，换来急需的钱。

改为：

> 主角必须在午夜前剪出一条能让观众迅速站队、转发、愤怒的舆论视频；他的阻碍不是素材不够，而是每一种剪法都会制造一个“看起来正确”的罪人。

这样更适合新结构。主角的动作不是“剪黑视频”，而是“剪出确定性”。这比单纯造谣更高级，也更危险。

## 4. 10 分钟四段式结构草案

### 第一幕：起因，约 1.5 分钟

开场钩子：黑屏里先听见笑声、停顿、某句“不舒服”。画面亮起，是一段综艺素材：男明星坐在人群边缘，其他人互相对视发笑。观众自然觉得他被排挤。

建置：剪辑师在深夜工作室里接到甲方消息：今晚必须出一版“能打”的片子，标题方向已经给好，情绪要明确，别中立。

激励事件：剪辑师打开素材库，里面有同一事件的多机位、道歉视频、网友玩梗、餐饮差评、男明星旧综艺争议片段。他开始剪第一版。

### 第二幕：发展，约 4 分钟

障碍 1：素材本身不够“锤”。很多瞬间都有双重解释：对视可能是尴尬，也可能是嘲笑；群聊可能是排除，也可能是包场；道歉可能是真诚，也可能是公关。剪辑师通过慢放、截断、反应镜头、弹幕覆盖，让它们变得更像“证据”。

障碍 2：甲方继续催，要更狠、更容易站队。剪辑师把网友玩梗、餐馆一星差评、父亲店铺冷清、男明星苦笑剪在一起。观众开始被推向明确立场：嘲笑者有罪，女明星有罪，网友有罪。

中点 / 伪胜利：剪辑师剪出一版很有力量的片子。它甚至不像营销视频，像一部正义短片。观众也会觉得“这片子说得对”。

一切尽失：剪辑师发现，如果把素材换一个顺序，也能剪出另一个版本：女明星被围攻，男明星粉丝反噬，网友借正义继续伤害另一个人。两个版本都成立。两个版本都能爆。

### 第三幕：高潮，约 2.5 分钟

最终对决：剪辑师在两个时间线之间切换。一版让观众同情男明星，一版让观众同情女明星；一版审判玩梗网友，一版审判粉丝反扑。素材相同，罪人不同。

关键选择：他没有剪“真相版”，而是剪出最有传播力的一版。不是因为他邪恶，而是因为他知道平台、甲方、观众都要这个：一个足够清楚的受害者，一个足够方便的加害者，一个可以转发时显得自己站在正义一边的情绪。

故事反转：成片播放。观众看到的，正是前 8-9 分钟这部短片本身。

### 第四幕：结局，约 2 分钟

画面退回剪辑软件时间线。刚才所有“自然发生”的段落都在轨道上：笑声被提前了半秒，沉默被拉长，女明星的解释被切掉后半句，男明星的苦笑被重复使用，餐馆差评声效被叠上去。

剪辑师没有发表宣言。他导出，上传，关掉屏幕。手机震动，数据开始涨。工作室里只剩下渲染完成的提示音。

最后一个画面：剪辑师淡淡离开位置。椅子空着，时间线还亮着。不是“真相揭晓”，而是“机器还在”。

## 5. 主角身份判断

推荐：**二者合并，但以“综艺后期出身、现在接短视频舆论单”为主。**

原因：

- 只做综艺后期剪辑师：更贴近素材源头，懂多机位、反应镜头、真人秀叙事制造，专业性强。但容易困在行业内幕，离“网友传播链”稍远。
- 只做短视频剪辑师：更贴近下沉传播、爆款逻辑、房租压力、普通人参与伤害链。但对综艺素材的操控专业性略弱。
- 合并最好：他曾是综艺后期，知道“现实感”怎么被制造；现在失业或转外包，靠短视频舆论单活着，知道“愤怒”怎么被变现。

这个身份能把两个主题缝在一起：**综艺制造关系，短视频制造审判。**

主角不是恶魔。他只是最适合这个时代的一颗螺丝：懂素材，缺钱，麻木，有手艺，也有一点点尚未死透的不安。
