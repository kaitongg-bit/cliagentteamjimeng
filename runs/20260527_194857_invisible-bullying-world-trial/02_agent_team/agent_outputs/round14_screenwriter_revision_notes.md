## 已完成的主要修改

1. **减少可读屏幕内容**
   - 删除或压缩了大量评论、截图、文件夹、声明段落的可读展示。
   - 屏幕更多作为光源、声音源、反射源和压力源出现。
   - 保留少数必要 UI 瞬间：项目名、`原始长镜头`、`暂不用`、`not_enough_to_convict`、协作记录、同步提示。

2. **弱化男明星旧黑料风险**
   - 移除 v9 中“旧综艺绊倒”作为旧黑料的强定罪段落。
   - 改为同场综艺游戏素材被误读：李云帝伸手、旁边人踉跄，工作人员“别挡机位”的上下文被切掉。
   - 保留“先制造男明星有罪”的效果，但不再给他编额外旧黑料。

3. **重新平衡女明星责任**
   - 孟青桐不是恶女，也不是全程无辜。
   - 她的冒犯来自：群聊里的轻慢、公开玩笑的迟钝、意识到可能过界后选择“算了”、社交优越感和节目现场里的微表情。
   - 第二轮定罪仍然成立为“可被剪辑放大”的素材，但完整素材会显示她不是恶意霸凌的单一反派。

4. **结尾冷处理**
   - 删除任何说教式总结。
   - 许临尝试做 `not_enough_to_convict`，但云盘协作记录显示系统已经复制、生成、发布。
   - 结尾落在空房间、自动保存、云端同步、手机外放“终于有人说了”，表达“人停下，系统继续”。

5. **生活动作与环境描写增强**
   - 增加冷馒头、外卖油印、旧台本、线缆、纸箱、节目徽章、清晨公交广告、楼道灯、电梯广告等可拍细节。
   - 空间按 Human 校准：普通一线城市旧小区单间，不再写成漏水危房或极端贫困。
   - 雨声保留为外部声源，主要落在窗外空调外机、玻璃、楼道和城市表面，不再用室内漏水/塑料盆卖惨。

6. **保持 7 场结构和 10 分钟体量**
   - 没新增主要角色。
   - 仍是：接单剪男方 → 发布扩散 → 接单剪女方 → 发布扩散 → 全素材打碎确定性 → 许临停下但系统继续。

## 仍需导演 / 细分镜处理的问题

1. **屏幕插入比例**
   - 剧本已减少可读屏幕，但细分镜还要进一步判断哪些 UI 信息改成反光、外放、手部停顿或门缝声音。

2. **同场游戏误读素材的可拍性**
   - 需要导演和分镜明确：李云帝伸手、男嘉宾踉跄、工作人员画外喊话，如何拍得暧昧但不变成新黑料。

3. **女明星“有错但非恶女”的表演尺度**
   - 孟青桐的微表情必须轻，不能演成反派白眼；她的“那算了”要像现实中的迟钝与懒得处理。

4. **第 6 场节奏**
   - 这一场需要明显慢下来，让观众从两轮定罪的速度里掉出来。细分镜应多用长镜头、少用剪辑爽感。

5. **许临最终动作**
   - 他把旧台本放进垃圾袋不是忏悔宣言，只是日常动作。导演需要避免拍成“洗白主角”的煽情节点。

6. **世界批判的冷感**
   - 不要让任何角色说出主题。结尾的批判应来自自动同步、外卖员外放、空房间和“终于有人说了”的重复。

- 单一核心事件：通过。核心仍是许临在一次急活中制造、复制、试图停止“定罪视频”。
- 10 分钟体量：通过。7 场结构保留，场景数量未膨胀。
- 可拍性：通过。动作、声音、光、空间细节为主。
- 红线：未写心理解释，未写说教独白。
- 伦理风险：男明星旧黑料已移除；女明星责任被保留但未恶女化。
- 结尾：通过。新常态为“人离开，项目仍在同步”。
