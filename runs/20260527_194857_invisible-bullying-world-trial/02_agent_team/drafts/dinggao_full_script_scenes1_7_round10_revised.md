# 《定稿》1-7 场完整剧本修订稿 round10

> 编剧修订 Agent 正式版。基于 round9 完整剧本、Showrunner 诊断、粗分镜 v1、视觉圣经 v1.1 与 moodboard round1 修订。
> 核心保留：先制造男明星有罪，再制造女明星有罪，最后更完整素材打碎确定性；许临停下，但系统继续。

【场景1：内景 许临出租屋 / 夜】

黑屏。

雨打在窗外的空调外机上。

咚。

鼠标点击声接上。

电脑亮起。冷光先照到桌面：一只普通显示器，一台轻薄笔记本，移动硬盘，耳机，外卖盒，纸巾，旧工牌，节目台本，几张揉皱的缴费单。

屋子不大。床在一侧，衣服搭在椅背上。铝合金窗框发灰，窗玻璃上挂着雨痕。墙面旧，但不是破屋。

许临坐在电脑前，三十二岁，胡茬没刮。深灰卫衣袖口起球。手机倒扣在键盘旁，震一下，屏幕光从桌面反上来。

甲方语音从手机外放漏出。

甲方语音：许老师，急活。今晚定，明早八点前要上。

许临把手机翻过来。房东消息压在语音下面，只露出半句。

房东消息：明早十点前。

许临把消息划走。

电脑里，一个压缩包解开。文件夹一排排铺开。许临没有点进去，只把鼠标停在最后一个文件夹上。

文件夹名一闪：

`原始长镜头`

甲方语音又来。

甲方语音：别中立。现在中立没人看。

许临点开素材。

监看窗口里，李云帝坐在综艺圆桌中间。舞台光干净，身边人笑得很近。孟青桐在他左侧，低头看了一眼杯子。

素材原声很远。

工作人员：云帝哥坐中间，我们机位好架。

许临把耳机戴上。工作人员的声音被盖住，只剩综艺笑声。

他新建项目。

项目名闪了一下：

`male_guilty_final_01`

他把李云帝低头的那一帧拖进时间线。

桌角有半个冷馒头。许临摸到它，又放下。

窗外一辆车压过积水，白光从玻璃上扫过，照出窗边灰尘和节目台本卷起的角。

许临打开标题框。光标在空白里闪。

甲方语音：观众点进来，三秒就得知道该骂谁。

许临输入一行标题。

标题盖住李云帝的脸。

他看了一眼，没有删。

雨声继续。

咚。

【场景2：内景 许临出租屋 / 夜深】

时间线被放大到帧。屏幕内容不清楚，只能看见波形、切点和许临眼睛里的蓝白反光。

李云帝的声音从耳机里漏出来。

李云帝：我确实有点不舒服。

下一秒，孟青桐的笑声接上。

许临按停。

他的食指停在鼠标上，指节有一小块翘起的创可贴。

他把笑声往前挪了半秒。

回放。

李云帝：我确实有点不舒服。

笑声立刻盖上来。

许临没有看屏幕。他看着窗玻璃里的自己。玻璃上，时间线变成一条很细的冷光。

甲方文字弹出。手机震动，把桌上的硬币震得转了一下。

甲方语音：有能接上的东西吗？别只剪一句，显得小题大做。

许临点开同场素材。综艺游戏环节里，李云帝伸手去扶旁边男嘉宾。画面晃，男嘉宾往后退了一步，现场一片笑。

另一机位里，工作人员在画外喊。

工作人员：快快快，往前走，别挡机位。

许临把工作人员那半句切掉。

他把晃动的一秒拖到“不舒服”后面。画面里只剩李云帝的手和旁边人的踉跄。

旧料文件夹停在素材栏边缘。

许临没有点开。

甲方语音：这个够了。同场更像真的。

许临拖进一条低频鼓。雨声被压低。

桌面上，旧台本被外卖汤盒压着。封面边缘沾了油，只能看见下方一行小字：

后期统筹 许临

许临把标题里的问号删掉。

屏幕白光跳了一下。

手机自动播放下一条语音。

甲方语音：再硬一点。

许临没有立刻动。他摘下一边耳机。

原始素材从耳机里漏出来。

李云帝：别拍成我扫兴。

工作人员：明白。

许临手停在空格键上。

门外有脚步声。楼道里有人刷短视频，外放声从门缝下挤进来。

手机视频声：一句不舒服，他到底忍了多久？

脚步声远了。

许临把原始长镜头缩进素材箱底部，改名。

`暂不用`

导出窗口弹出。文件名自动填好。

许临把鼠标移到“导出”。

停了一下。

窗外雨滴又打在空调外机上。

咚。

他点下去。

【场景3：内景 许临出租屋 / 小区门口 / 夜】

导出完成的提示音很轻。

许临把视频拖进发布后台。封面自动停在李云帝低头、孟青桐笑的那一帧。

他点击发布。

电脑发出两声系统提示。

许临咬了一口冷馒头。馒头边缘掉下一点屑，落在键盘缝里。

评论没有正面出现在画面里。只有屏幕光在许临脸上变快，像有人把雨点打在他眼睛里。

几句声音叠起来。

评论声一：终于有人说了。

评论声二：强者卖惨。

评论声三：他爸那个店是不是也在这边？

许临的咀嚼停了一下。

切到——

【外景 小区门口雨棚 / 夜】

雨棚边缘滴水。外卖员坐在电动车上等单，手机夹在车把上。竖屏视频里，标题挡住李云帝半张脸，评论区盖住孟青桐的笑。

他没有认真看。拇指往上滑，嘴里嚼着便利店饭团。

旁边一个女人收伞，瞟了一眼。

女人：又谁？

外卖员：不知道。热搜。

绿灯亮。电动车冲出去，手机外放还在响。

李云帝：不舒服。

切回——

【内景 许临出租屋 / 夜】

门铃响。

许临开门。外卖员站在楼道里，雨水从头盔边滴到外卖袋上。他手机还亮着，正播同一条视频。

外卖员扫了一眼屋里的电脑。

外卖员：你也看啊。

许临接过袋子。

许临：嗯。

外卖员：这男的挺会的。

许临没接话。

外卖员低头看手机。

外卖员：评论说他爸店被打差评了。现在人真闲。

楼道灯灭了。

外卖员伸手拍了拍墙上的开关。灯亮，白光硬硬地切过许临的脸。

外卖员：给个好评啊。

许临点头。

门关上。

外卖袋放到桌上，汤盒边缘渗出一点油。油蹭到缴费单上，留下一圈透明印。

手机震动。

甲方语音：可以。先别停。

又一条语音。

甲方语音：评论转向了。有人扒女方那边。你接着做。

新的压缩包开始下载。

文件名只露出前几个字：

`孟青桐...`

许临抽纸擦油。纸巾越擦越薄，油印还在。

下载进度条往前爬。

窗外雨声小了一点，楼道里的短视频声变得更清楚。

【场景4：内景 许临出租屋 / 后半夜】

压缩包解开。

许临新建项目。

项目名一闪：

`female_bully_final_01`

第一段是综艺圆桌。李云帝说完一句话，低头喝水。孟青桐和旁边女嘉宾对视了一下。

女嘉宾用口型说了两个字。

云底。

孟青桐没有立刻笑。她低头，嘴角动了一下。

圆桌另一边有人先笑出声。

李云帝抬眼。

许临把现场笑声提亮。

他没有加混响。

回放时，那一下低头被拖长。

甲方文字弹出。

甲方文字：她这个表情很准。

许临点开颁奖素材。

舞台很亮。孟青桐拿着奖杯。

孟青桐：谢谢大家，也谢谢云帝哥。

台下有人起哄。

观众：是云底吧！

台上笑开。孟青桐也笑了一下，看向李云帝那边。

孟青桐：那今天就叫云底哥吧。不是那个帝，是底下托着我们的底。

许临按停。

后半句还没出来，掌声已经起了。

他把后半段拖到素材箱边上。

文件名：

`暂不用_后半句`

另一个素材里，后台侧光很白。孟青桐拿着奖杯下台。

孟青桐：我刚才那个，是不是有点过？

助理：没事，大家都笑了。

孟青桐看向台口。李云帝正在跟品牌方合影，脸上挂着笑。

孟青桐没有再过去。

她把奖杯递给助理。

孟青桐：那走吧。

许临把这段也拖到边上。

甲方语音：这个别放。她要是知道不妥还没补，观众会分散。我们就打她轻慢。

许临点开群聊截图。画面只给电脑屏幕侧面，观众看不清长文字，只看见许临用触控板把几行往上拖。

手机外放念出被语音识别的一句。

语音识别声：先小群定吧，省得他又不舒服。

许临把这一句裁出来。

下一行短短闪过。

`回头单独约他`

他停了一下，把后一行拖出画面。

甲方文字：够了，别给她台阶。

许临看着时间线。

他点开另一个素材：主持人让孟青桐说两句。孟青桐低头看手机。另一位嘉宾接话。

全机位里，手机屏幕反光很亮，像一块白斑。

许临放大那块白斑。工作群内容看不清，只能听见素材里主持人笑着转场。

主持人：那我们先下一位。

许临把孟青桐低头的一秒拉长。

他打出一行标题。问号停在末尾。

他删掉问号。

房间里没有鼓声，只有键盘声。

外卖汤已经冷了，盒盖上的油凝成一圈白边。

许临把低频鼓拖进来。

鼓声慢慢压住雨声。

【场景5：内景 许临出租屋 与 楼道 / 凌晨】

许临的手指悬在触控板上。

手机连震两下。

房东消息的亮光一闪。

甲方文字的亮光跟着一闪。

他把第二条视频拖进发布后台。

封面自动停在孟青桐低头、李云帝抬眼的那一帧。

许临点发布。

两声系统提示后，房间更静。

他打开两条视频后台。镜头不拍字，只拍两个窗口并排发出的白光，拍许临的眼睛在左右移动。

评论声从左右声道叠起来。

左声道：终于有人说了。

右声道：终于有人说了。

左声道：他不是第一次这样。

右声道：她不是第一次这样。

左声道：别装无辜。

右声道：别装无辜。

许临把音量关小。

门外，电梯到了。

外卖员拎着两份餐，停在隔壁门口等人开门。他的手机外放是孟青桐那条视频。

孟青桐：那今天就叫云底哥吧。

鼓声压上来。

外卖员：外卖。

隔壁门开一条缝。一只手伸出来。

邻居：前两天不是男的吗？

外卖员：都一样。

他把餐递过去，顺手点了个赞。

外卖员：给个好评啊。

电梯门合上。

许临打开门。

楼道灯坏了一盏，剩下那盏闪。电梯门上贴着孟青桐代言的生活服务广告。照片里她笑得干净。

广告旁边被人用黑笔写了两个字：

道歉

字被雨衣袖子蹭花了。

许临站了一会儿。

手机在屋里响。

甲方语音从电脑外放里自动播出来。

甲方语音：她肯定要回应。回应前我们再补一条。

许临回到屋里。

电脑后台提示音连续响。他把窗口关掉，声音停了一秒，又从另一个自动播放的小窗里冒出来。

评论声：她家那个花店是不是也搜得到？

许临把音量按到底。

屏幕还是亮着。

他低头看地上的线。线缆缠在桌脚和移动硬盘之间。

他蹲下去，把插线板往外挪了挪。

桌面上，旧台本被外卖袋压住。油印从塑料袋底慢慢渗到封面上。

许临抽出一张纸，垫在下面。

纸很快透明。

【场景6：内景 许临出租屋 / 凌晨将尽】

雨停了一阵。

屋里没有开灯。电脑屏幕上是全机位素材，声音很干净，没有鼓。

许临坐着，手没有放在鼠标上。

圆桌全景里，主持人问李云帝。

主持人：你刚才说不舒服，是身体不舒服，还是这个环节不舒服？

李云帝拿着水杯，杯口碰到嘴边，又放下。

李云帝：都有点吧。录太久了。

主持人笑。

主持人：青桐，你们认识最久，你说两句？

孟青桐低头看手机。

全机位里能看见工作人员在画外递台本。台本角上夹着一张流程改动。

孟青桐抬头慢了半拍。

孟青桐：他今天确实累了。

主持人已经转向别人。

许临没有按暂停。

他点开李云帝原始长镜头。

李云帝坐在圆桌主位，摄像机重新调焦。

李云帝：我不想玩那个互相整人的环节。别拍成我扫兴。

工作人员：明白。

旁边有人递来台本。

李云帝翻了两页。

李云帝：孟青桐那段别剪太满，她容易被骂。

工作人员：这个看后期。

李云帝没再说。

许临点开孟青桐后台花絮。

孟青桐：我刚才那个，是不是有点过？

助理：没事，大家都笑了。

孟青桐看向台口。李云帝和品牌方合影，笑挂在脸上。

她站了两秒。

孟青桐：那算了。

助理：车到了。

孟青桐走了。

电脑风扇声变得很清楚。

甲方语音弹出。许临没有点。

他在聊天框里输入：

`全素材不够定她。`

停了一下。

又输入：

`也不够定他。`

发送。

对面很快回。

甲方文字：

`我们不是法院。`

第二条接上。

`尾款还要不要？`

许临的手机亮起。银行卡到账提醒弹出一半，被低电量提示盖住。

他摸到电源线，插上。

线不够长。他弯腰把桌脚旁的插线板往外拉。插线板卡在纸箱后面。

纸箱侧面写着：公司私人物品。

许临把箱子往旁边推。

里面的旧工牌、几根线、一个节目徽章滑出来。

节目徽章滚到床脚，停住。

许临坐回去。

他新建项目。

项目名：

`not_enough_to_convict`

他把两个原始长镜头拖进去。

时间线空了很久，才出现第一段完整声音。

没有红圈。

没有鼓。

没有字幕。

天亮得很慢。

窗玻璃发白。远处第一班公交从楼下过去，车身广告是孟青桐的脸。雨水把颜色洗淡，笑容变得模糊。

许临还坐在电脑前。

导出窗口打开。

文件名：

`not_enough_to_convict_v01`

鼠标停在导出按钮上。

手机震动。

房东消息：

`钱收到了。下个月别拖。`

甲方消息跟着进来。

`你那个不用发了。`

下面是一张后台截图。标题看不清，只能看见孟青桐低头的封面、红色标注和发布成功的绿点。

`03:41 成员A 复制项目`

`03:42 成员A 生成封面`

`03:44 成员A 发布成功`

后半句被他自己的手挡住。

他点取消。

甲方语音自动播放。

甲方语音：许老师，不是针对你。窗口就这么点，谁慢谁没饭吃。

许临摘下耳机。

屋里安静下来。

另一个视频自动续播。陌生男声从电脑外放里传出来。

陌生男声：这两条视频，源头找到了。

许临按暂停。

画面边缘有一条竖着的暗缝，像门没关严。

许临看向门。

门关着。

他站起来，把外卖袋扎好。透明盒盖上的油晃了一下。

他拿起旧台本。

封面被油印浸出一块深色，节目名糊了，下面一行小字还在。

后期统筹 许临

许临把台本放进垃圾袋。

电脑屏幕上，`not_enough_to_convict_v01` 的导出窗口还开着。

按钮仍然亮。

许临关掉台灯。房间暗下来，电脑更亮。

他拎着垃圾袋走到门口，停一下，回头看。
