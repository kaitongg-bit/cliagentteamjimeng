# 《定稿》1-7 场完整剧本初稿

> 当前合并版本：第 1-2 场来自 round4；第 3 场来自 round5；第 4 场来自 round6 修订版；第 5-7 场来自 round8。
> 这是供审片人 Agent 与导演 Agent 会诊的工作稿。

【场景1：内景 许临出租屋 / 夜】

黑屏。

雨声先到。

很近。像有人用指甲刮着铁皮。

一滴水落进塑料盆。

咚。

一声鼠标点击。

剪辑软件亮起。时间线密密麻麻，项目名停在左上角：

`male_guilty_final_02`

播放窗口里，李云帝坐在综艺圆桌正中。身边几个人笑着接话。画面定格在他低头的一帧。

电脑屏幕的光照出半间屋子。

窗缝被胶带贴过，胶带边缘翘起。雨水从翘起处渗进来，沿墙皮往下爬。墙角的塑料盆里已经积了浅浅一层水。

旧工牌挂在椅背上，卡套里只剩一张过期门禁。旁边压着一叠打印纸，最上面露出四个字：离职交接。

桌下的插线板垫在两本旧书上，书角被泡软。

手机震动。

屏幕右上角弹出房东消息：  
`明早十点前。别让我再催。`

许临坐在电脑前，三十二岁，胡茬没刮，外卖袋压着半张欠费单。他把消息划掉，指尖停在空格键上。

他的食指指节贴着一片创可贴，边缘发黑。鼠标垫上有一圈咖啡渍，形状像干掉的水位线。

又一条消息进来。甲方头像没有名字，只有一个灰色字母。

甲方语音：许老师，急活。今晚定。明早八点前要上。

许临点开尾款截图。数字不大，但够房租。

他把截图拖到桌面角落。桌面上已经有几个没来得及删的文件夹：

`简历_第七版`  
`后期作品集_压缩`  
`不要再投这个公司`

他把耳机戴上，拉近时间线。

许临：方向？

甲方语音：别中立。现在中立没人看。

许临新建一条字幕轨。光标在标题框里闪。

甲方语音：观众点进来，三秒就得知道该骂谁。

许临删掉标题框里的空格。

许临：素材全吗？

甲方发来一个压缩包。

文件名：  
`李云帝风波_可用素材_pack`

许临下载。进度条跑得很慢。房东消息又弹出来，盖住进度条一角。

窗外一辆车压过积水，白光从窗帘底下扫过去。屋里短暂亮了一下，又暗回去。

塑料盆又接住一滴水。

甲方语音：先走男的。强者卖惨，粉丝下场，女方被网暴。你干过综艺，知道怎么剪。

许临没回。他把房东消息拖到屏幕边缘。

压缩包解开。

文件夹一排排展开：

`01_主位`  
`02_他说不舒服`  
`03_众人陪笑`  
`04_粉丝控评`  
`05_旧综艺绊倒`  
`06_原始长镜头`

鼠标停在 `06_原始长镜头` 上。

停了两秒。

甲方又发来一条文字：  
`标题先定，别被素材带跑。`

许临点开 `01_主位`。

监看窗口里，李云帝坐在桌子正中，灯光打得最亮。孟青桐在他左侧，低头笑了一下。

许临把这一段拖进时间线。

拖拽时，鼠标线碰到桌边一枚硬币。硬币滚了半圈，停在欠费单上。

他点开标题框，输入：

`他一句“不舒服”，毁了别人半生？`

标题盖住李云帝的脸。

许临看了一眼窗边。

塑料盆快满了。他抽出一张旧综艺台本封面，折了两下，垫到窗台下方，让水顺着纸面流进盆里。

台本封面上印着节目名，下面一行小字：后期统筹 许临。

他回到电脑前。

手机屏幕暗下去。房间里只剩电脑风扇声和素材里的掌声。

许临按下保存。

项目名变成：

`male_guilty_final_03`

切。

【场景2：内景 许临出租屋 / 电脑时间线 / 夜】

雨更密。

窗玻璃上有城市的倒影，霓虹被雨水拉长。远处一块商场大屏轮播招聘广告，字太亮，看不清职位，只看见“年轻”“热爱”“抗压”几个词反复闪。

时间线放大到帧。

第一段：李云帝坐主位。  
第二段：众人笑。  
第三段：李云帝对镜头说“我确实有点不舒服”。

许临把第三段往前拖了半秒。

李云帝：我确实有点不舒服。

下一帧立刻接上孟青桐的笑。

许临回放。

李云帝：我确实有点不舒服。

孟青桐笑声响起。

许临按暂停。切点落在孟青桐抬眼前一帧。

他右手没有离开鼠标，左手摸到桌边的冷馒头。馒头被雨夜的潮气泡软，掰开时没有声音。

他把笑声复制，贴到前面一点。再回放。

笑声提前了。

李云帝低头的四秒变长。

许临打开音乐库，试听几条音轨。

轻快鼓点。

他删掉。

低频鼓声。

他拖进音轨，压在李云帝低头那四秒下面。

监看窗口里，李云帝的沉默变沉。字幕弹出：

`他早就不满？`

许临把问号删掉。

字幕变成：

`他早就不满。`

微信弹出甲方文字：  
`别问句，问句软。`

许临把馒头放回塑料袋。塑料袋口没有扎紧，里面的水汽贴在袋壁上。

许临没回。他点开 `05_旧综艺绊倒`。

旧综艺画质更糊。李云帝在竞技游戏里伸手，旁边男嘉宾摔倒。现场笑成一片。

许临把时间戳裁掉，调成和前面一样的冷色。

屏幕上出现新标题：

`不是第一次了`

他把旧综艺片段接在“不舒服”后面。

又点开 `04_粉丝控评`。

一张张截图快速铺进时间线：

`哥哥只是说实话`  
`孟青桐出来道歉`  
`别装无辜`  
`品牌还不撤？`

许临把截图速度调快。评论像雨点一样闪过。

他又拖进一张热搜截图：

`李云帝 不舒服`

热搜后面跟着一个红色“爆”。

许临放大红字，裁掉截图右下角日期。

窗边的塑料盆轻轻晃了一下。楼上有脚步声，水管里传来一阵空响。

许临拉开抽屉找东西。抽屉里有半盒感冒药、两张未报销的出租车票、一枚节目组纪念徽章。

他没找到胶带，只把抽屉推回去。

回放。

李云帝坐主位。  
众人笑。  
李云帝低头。  
低频鼓声。  
“不舒服”。  
旧综艺摔倒。  
粉丝截图。  
热搜爆字。

许临摘下一边耳机。素材原声从另一边漏出来。

原始素材里，有人喊：

节目工作人员：云帝哥，您坐中间，我们机位好架。

许临按空格。

画面停在工作人员伸手指座位的一帧。

雨声短暂压过耳机漏音。

屏幕上，工作人员的手指悬在李云帝肩后。画面边缘有一只没收进镜头的麦克风。

他把这一小段拖到素材箱底部，改名：

`暂不用`

甲方语音弹出。

甲方语音：有旧料吗？别只剪这次，显得小题大做。

许临看着 `暂不用` 的缩略图。

他点开旧料文件夹，把摔倒片段又复制了一遍。

标题轨上，新的大字压下来：

`他把“不舒服”变成武器`

许临拖动音量线，把李云帝原声压低，只留下“我不舒服”三个字更清楚。

李云帝：我……不舒服。

孟青桐的笑声接上。

许临把笑声加了混响。

房间门外，有邻居拖行李箱经过。轮子声短短擦过地面。

楼道灯从门缝下亮起，又灭掉。

许临脚边，一只纸箱被雨水泡塌了角。箱子侧面写着：公司私人物品。

许临没有抬头。

导出窗口弹出。

文件名自动填入：

`male_guilty_final_03_REALFINAL.mp4`

他把鼠标移到“导出”。

停住。

时间线底部，`暂不用` 的缩略图还在。李云帝坐在桌边，工作人员的手停在半空。

许临把导出窗口挪到一边，露出缩略图。

甲方文字跳出：  
`先发，热度窗口快过了。`

许临点下“导出”。

进度条从 1% 跑起。

许临起身去倒塑料盆。

他端起盆，水面晃出电脑屏幕的光。盆底有一小片墙皮。

他走到卫生间门口，停住。

耳机线还挂在脖子上，低频鼓声从耳机里漏出来。

他把水倒进洗手池。下水口堵了一下，水慢慢旋下去。

回到桌前时，导出已经快结束。

低频鼓声还在耳机里循环。  
李云帝的声音被切成很短的一句：

李云帝：不舒服。

进度条到 100%。

【场景3：内景 许临出租屋 / 发布后台 与 城市雨夜 / 夜】

导出完成提示音。

文件停在桌面：

`male_guilty_final_03_REALFINAL.mp4`

许临把视频拖进发布后台。

封面自动生成：李云帝低头，孟青桐的笑停在他身后。

标题栏里已有一行字：

`他把“不舒服”变成武器`

许临看着封面。

窗台下的旧台本被水泡软，纸面塌下去一点。水没有再顺进盆里，沿着墙往下流。

手机震动。

房东消息：  
`十点。`

甲方文字：  
`发。`

许临点下发布。

后台弹出：

`审核通过`

`发布成功`

房间里很静。只有雨声和电脑风扇声。

几秒后，评论开始刷新。

`终于有人说了`  
`强者卖惨最恶心`  
`孟青桐被他粉丝骂成那样，他还装？`  
`他爸不是开店吗`

许临把后台窗口缩小，点开收益预估。数字跳了一下，很小。

他拿起桌边的冷馒头，咬了一口。

视频声音从电脑里自动播放。

李云帝：我……不舒服。

低频鼓声接上。

切到——

【外景 小区门口 / 夜】

雨棚下，外卖员把手机夹在电动车把手上，等红灯。

同一条视频在竖屏里播放。标题挤在李云帝脸上。

外卖员拇指一划，评论区盖住画面。

他笑了一下，把手机音量调大。

旁边一个穿雨衣的女人看了一眼屏幕。

女人：又谁塌了？

外卖员：不知道。热搜第一。

绿灯亮。

电动车冲进雨里。手机里还在响。

李云帝：不舒服。

切回——

【内景 许临出租屋 / 夜】

后台曲线往上抬。

评论刷新更快。

`李记汤面是不是他爸开的？`  
`去打分啊，别光骂`  
`一家人都吃红利吧`

许临的鼠标停住。

他点开一张评论截图。截图里是一家小餐馆的页面，店名：李记汤面。评分旁边的星星正在变少。

新差评一条条冒出来。

`儿子这样，家里也不会好`  
`不吃了`  
`退单`

许临把截图关掉。

门铃响。

他起身开门。

门外是外卖员，雨水顺着头盔边往下滴。外卖袋上贴着促销券。

外卖员把袋子递给他，手机还亮着，正播许临刚发的视频。

视频里，孟青桐笑声被混响拉长。

外卖员扫了一眼许临屋里的电脑。

外卖员：你也看这个啊。

许临接过袋子。

许临：嗯。

外卖员：这男的挺会的。

许临没接话。

外卖员看手机。

外卖员：哎，评论说他爸店被冲了。

楼道灯灭。

外卖员按了一下墙上的开关，灯又亮。许临的脸被白光照了一下。

外卖员：给个好评啊。

许临点头。

关门。

外卖袋放到桌上，袋底蹭过一张欠费单。汤洒出一点，油顺着塑料盒边缘流到桌面。

电脑提示音连续响。

甲方发来语音。

甲方语音：可以。先别停。

许临坐回去。

甲方又发来一条。

甲方语音：评论区开始转了。有人扒孟青桐那边。你接着做。

许临看着发布后台。视频封面上的李云帝被弹幕一层层盖住。

甲方发来新压缩包。

文件名：

`孟青桐_隐形霸凌素材_pack`

下面跟一条文字：

`现在大家想看这个。`

许临没有动。

窗台上的旧台本彻底塌下来，水直接滴到地板上。

咚。

咚。

咚。

甲方文字：  
`热度窗口还在。`

许临拖过纸巾，擦掉桌上的汤油。纸巾越擦越花。

他点下下载。

进度条开始跑。

【场景4：内景 许临出租屋 / 素材库新文件夹 / 夜】

压缩包解开。

文件夹铺满屏幕：

`01_群聊不带他`  
`02_对视笑`  
`03_颁奖词_云底`  
`04_不接话`  
`05_道歉声明草稿`  
`06_旧直播切片`  
`07_多机位原始`

许临新建项目。

项目名闪了一下：

`female_bully_final_01`

他点开 `01_群聊不带他`。

一张聊天截图打开。群名被红框圈住：

`青桐生日局`

成员列表里没有李云帝。

许临放大群名。

群聊下方几行字露出来：

`别拉他吧，他来了大家都要围着他转。`

`老板来了不像朋友局。`

`他会不会又说自己不舒服哈哈。`

最后两行是孟青桐发的。

`别在大群说。`

隔了几秒。

她又发：

`先小群定吧，省得他又不舒服。`

许临把群聊往上拖，只留最后一句。

屏幕里只剩：

`先小群定吧，省得他又不舒服。`

甲方文字弹出：

`这个好，够准。`

许临没有回。

他点开 `02_对视笑`。

监看窗口里，综艺圆桌。李云帝说完一句话，低头喝水。

孟青桐和旁边女嘉宾对视。

女嘉宾用口型说了两个字：

`云底`

孟青桐先没笑。

下一秒，她低头，嘴角动了一下。

圆桌另一边有人笑出声。

李云帝抬眼。

笑声停得很快。

许临把这一段拖进时间线。

他没有加混响。

只把现场笑声提亮。

回放。

李云帝抬眼的那一帧被停住。

许临加字幕：

`他听见了。`

他看了一眼，删掉。

改成：

`他们都听见了。`

手机震动。

房东消息：

`收到钱我就不说了。`

许临把手机扣过去。

他点开 `03_颁奖词_云底`。

舞台灯光很亮。孟青桐拿着奖杯，身边站着几个嘉宾。

孟青桐：谢谢大家，也谢谢云帝哥。

台下有人笑，有人喊了一声。

观众：是云底吧！

台上几个人笑。孟青桐也笑了一下。

孟青桐握着话筒，看向李云帝那边。

李云帝也笑了一下。

孟青桐：那今天就叫云底哥吧。不是那个帝，是底下托着我们的底。

台下掌声和笑声混在一起。

李云帝脸上的笑慢慢收掉。

孟青桐继续说：

孟青桐：因为他一直在后面托着大家。

许临按空格。

画面停在“底下托着我们的底”。

他把后半句拖到素材箱底部。

改名：

`暂不用_解释`

甲方语音弹出。

甲方语音：这个就是爆点。别解释，一解释就软。

许临把“云底”那一句复制到时间线。

字幕：

`当众改名`

他又加一行：

`只是玩笑吗？`

问号停了两秒。

他删掉问号。

`只是玩笑。`

许临盯着这四个字。

删掉。

改成：

`不是玩笑。`

雨水从窗台滴到地板上。

咚。

他点开 `04_不接话`。

素材里，主持人看向孟青桐。

主持人：青桐，你和云帝认识最久，你说两句？

孟青桐低头看手机。

李云帝抬眼看她。

另一位嘉宾接话。

许临把孟青桐手机屏幕放大。

屏幕上是节目组工作群：

`下一轮流程改了，青桐看手机确认。`

他把这三秒往前拖出去。

只剩主持人问话后，孟青桐低头。

李云帝抬眼。

另一位嘉宾接话。

许临加红圈，圈住孟青桐的眼睛。

红圈跟不上她低头的动作，抖了一下。

他重新打关键帧。

红圈稳住。

门外有脚步声经过。

手机外放传进来：

手机视频声：男明星一句不舒服，到底忍了多久？

脚步声远去。
