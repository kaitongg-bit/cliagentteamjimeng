# Clean-Context Single Agent Baseline Notes

## 定位

这是在 clean context 下完成的 single agent baseline。创作仅基于以下三份输入：

- `00_brief/brief.md`
- `00_brief/user_story_seed.md`
- `01_single_agent/baseline_prompt.md`

未读取、未引用 agent team 后续产物，也未使用本轮对话之外的团队创作结论。

## 创作取舍

1. **审判对象从个体转向机制。**  
   剧本没有把李云底写成绝对受害者，也没有把周芒写成绝对加害者。两人都被节目机制、热搜机制、职业焦虑和观众情绪推着走。真正被呈现的是“每个人都只做了一点点，最后伤害变得很完整”。

2. **保留事件种子的识别度，但不做复盘。**  
   “包场群”“拖底奖杯”“不舒服文学”“父亲店铺差评”等元素被保留为戏剧支点，但不追求逐条解释现实事件，而是压缩成一个可拍的核心事件：一句真实感受被平台加工成公共笑料。

3. **把 AI 时代的不可替代性放进环境。**  
   brief 中“AI 时代失去不可替代性”的主题通过透明屏 AI 剪辑推荐、舆情面板、数字人广告来呈现。李云底面对的不只是网友，也是一个永远得体、永远可复制的“年轻版自己”。

4. **主要角色控制在三人。**  
   李云底、周芒、老李承担核心弧光。其他嘉宾、食客、骑手、女孩只作为舆论生态的切片出现，不发展独立人物线。

5. **场景控制在五个。**  
   综艺棚、后台、面馆、酒店、清晨面馆形成从公共表演到私人承受再回到公共空间的闭环。这样有利于 10 分钟微电影拍摄，也符合主要场景不超过 5 个的限制。

6. **反转不做反杀。**  
   第四场出现路人视频和部分评论松动，但没有安排男主或女主“赢回舆论”。结尾只是少数人把手机扣下、来店里吃面，改变很小，但可被画面看见。

7. **教育意义通过动作实现。**  
   片尾手写纸条、扣下的手机、女孩删除一星、周芒转发路人视频，都是具体动作。尽量避免角色直接讲“网络暴力不好”。

## 视觉记忆点

- 综艺棚透明屏把“苦笑”“不舒服”即时标注成爆点。
- 面馆打印机吐出一长串一星差评小票，差评实体化。
- 片尾一排手机屏幕朝下，像审判席暂时闭嘴。

## 未解决风险

1. **议题密度偏高。**  
   10 分钟内同时处理隐形霸凌、饭圈/黑营销、经济下行、AI 替代焦虑，存在信息拥挤风险。拍摄时需要依靠节奏和留白控制，不宜让屏幕文字过多。

2. **女主周芒弧光较克制。**  
   她的转变主要体现为来酒店、承认害怕、转发路人视频。若希望女性角色更强，后续可增加她在节目现场主动停下笑声的回忆或补拍动作。

3. **结尾的善意人群可能被理解为“舆论洗白”。**  
   目前处理为少数人安静到店、没有大和解。拍摄时应避免煽情音乐和大规模排队，以免削弱对机制的批判。

4. **“网友”无法具象成单一反派。**  
   这是有意选择，但也意味着戏剧冲突不如单一反派明确。需要靠差评小票、弹幕、热搜面板等可视化装置增强压迫感。

5. **现实指涉需保持距离。**  
   剧本使用了事件种子的结构元素，但角色、节目名、情节走向均为虚构。实际制作时仍应避免真实姓名、真实节目视觉和可识别口头禅。
