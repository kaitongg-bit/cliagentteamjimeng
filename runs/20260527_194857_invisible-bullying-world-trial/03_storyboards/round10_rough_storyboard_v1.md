# 《定稿》粗分镜 v1

## 全片视觉策略

全片不把屏幕当“信息板”，而把它当光源、声源、反射源和压力源。观众需要知道的信息，只保留项目名、标题压脸、问号变句号、发布成功、同步提示等少数关键屏幕瞬间；评论和热搜尽量用声音叠化、脸被遮挡、外放漏音、楼道传播来完成。

水是节拍器：从塑料盆一滴水，到旧台本导水，到台本塌掉，水流向插线板，最后回到黑屏雨声和一滴水。许临的身体动作从“快速剪”“快速点”“端盆处理”，逐渐变成“不点语音”“不剪”“踢开插线板”“收垃圾离开”。

空间扩散路径：出租屋电脑光 → 耳机漏音 → 发布后台 → 小区雨棚竖屏 → 楼道门缝外放 → 电梯广告涂写 → 城市公交广告 → 空房间自动同步。舆论不是被解释出来，而是从一个屏幕渗到整栋楼、整座城市。

## 给细分镜 Agent 的提醒

少拍可读屏幕，多拍屏幕造成的光、声音和停顿。每个屏幕插入都先问：能不能改成耳机漏音、眼睛反光、评论外放、门缝传声、水面倒影、手指停住。

旧料段落务必降权。男方旧综艺和女方旧直播不要拍成“新证据”，只拍成许临素材库里被加工、被拖拽、被声音污染的东西。

第5场楼道是第二主场，不是过场。门缝、坏灯、电梯广告、外卖员外放，比两个后台窗口更重要。

第6-7场要明显慢下来。不要用惊悚音乐，不要用突然推近，不要让许临说主题。最后的情绪应该落在：人走了，项目还在同步。
