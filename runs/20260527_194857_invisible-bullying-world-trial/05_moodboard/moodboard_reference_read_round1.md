# Moodboard Reference Read Round 1

本轮输入来自 Human 主动找图，不把任何参考图当成要复制的最终画面，只抽取可迁移的镜头规则、空间规则、表演规则和负面约束。

## 总体判断

这批图把《定稿》的视觉方向从“AI 生成的脏电影感”拉回了更准确的方向：普通空间、平台截图、综艺素材、劳动者环境、吃瓜场景。

最重要的 Human 判断已经出现：

- 许临的家不是极端破烂，也不是漏水苦难空间，而是一线城市里弄/老旧小区里比较不打理的普通住处。
- 许临的电脑很正常：普通显示器、MacBook、正常当代办公设备都可以；不要复古 CRT，也不要刻意低配。
- 许临空间最重要的是《宇宙探索编辑部》那种影调和生活质感：普通、旧、冷、闷、有点狼狈，但不卖惨。
- 综艺段落应该明亮、干净、可消费，和许临的出租屋形成强对比。
- 男明星被孤立的素材要靠人物距离、站位、背后小团体来制造不适，而不是给男明星编黑料。
- 父亲餐饮线要有普通城市早起劳作的环境侧写，不要把父亲拍成苦情工具人。
- 普通人吃瓜不是宏大舆论隐喻，而是餐厅、手机、饭桌、等待时间里的低成本注意力流动。

## 参考图逐张解读

### IMG_0001 / 宇宙探索编辑部.png

Human 标签：只要空间。

Human 备注：许临的家大概这样，不是特别破烂的。

可学习：

- 房间不豪华，但也不是“贫民窟式惨”。这很重要，避免 AI 把许临家画成极端苦难。
- 空间里有床、桌、柜、杂物、旧墙面，生活功能混在一起。
- 人被空间包住，而不是站在一个为了他服务的漂亮场景里。
- 影调和质感比具体道具更重要：冷、灰、旧、湿闷，但不是漏水危房。

不要学：

- 不要照搬电影本身的人物造型和年代质感。
- 不要让房间过度复古或过度荒诞。
- 不要把“低质量真实”误解成穷、破、漏水、CRT 电脑。

转成镜头规则：

- 许临出租屋可以是中全景/全景，先让观众读到“空间如何吞人”。
- 房间质感：一线城市老旧小区/里弄单间，普通旧、乱、不打理，但仍是正常生活空间。
- 电脑规则：正常显示器或 MacBook，屏幕作为光源；不要 CRT、不要复古电脑房、不要黑客工作站。
- Prompt 负面约束要加：不要废墟感、不要年代戏、不要极端贫困景观、不要漏水苦难。

### IMG_0002 / 许临侧脸.png

Human 标签：只要景别、只要光线、只要影调、只要质感。

Human 备注：看到许临在这个家工作的背影后，可以切一个这样的镜头，侧脸，眼睛反光（电脑的光）。

可学习：

- 这是一个从空间进入人物内心的切换：先背影/空间，再侧脸/眼睛反光。
- 侧脸近景可以让电脑光进入眼镜或眼睛，让“屏幕正在塑造他”可视化。
- 质感可以偏湿、闷、近距离，皮肤不需要美化。

不要学：

- 不要把他拍成英雄式忧郁肖像。
- 不要过度戏剧化眼泪或崩溃。

转成镜头规则：

- S01 可拆成两个镜头：门口中全景建立空间，随后侧脸近景捕捉屏幕光反射。
- 侧脸镜头里电脑内容不必可读，重点是光和眼睛的关系。

### IMG_0003 / 综艺截图.png

Human 标签：只要站位、只要空间、只要景别、只要影调、只要光线。

Human 备注：综艺明亮的光，人物关系，综艺视频还会切一些近景和配一些字，但是大概是这种环境和人物站位，男女比例。

可学习：

- 综艺空间是明亮、干净、偏消费级的室内，和许临的房间形成强反差。
- 人物关系可以通过站位读出来：男明星处于“被观看/被隔开”的位置，小团体在同一块区域形成亲密。
- 综艺画面天然带有监控/机位感，可以作为许临剪辑素材。

不要学：

- 不要复制真实综艺品牌、Logo、字幕样式。
- 不要把综艺空间拍得太高级，应该是平台综艺的可消费、可剪辑、可截屏空间。

转成镜头规则：

- 综艺素材可用俯一点的广角/固定机位建立群体关系。
- 后续剪辑可以切局部近景，让“互看、憋笑、沉默”被恶意放大。

### IMG_0004 / 综艺截图暂停键画面.png

Human 标签：未标注。

Human 备注：学综艺视频的暂停键，还有后期的那些如“试图理解中”等的标注性的文字、图案。

可学习：

- 这一张不是学人物，而是学“平台素材被暂停、被截取、被加字”的状态。
- 播放按钮、暂停画面、花字可以成为许临剪辑过程里的操控痕迹。

不要学：

- 不要要求生图模型生成准确文字。文字、暂停键、标注图案应该后期合成。

转成镜头规则：

- Prompt 里只写“综艺暂停帧/大块图形标注占位”，具体中文字由后期 UI 合成。
- 剪辑 Agent 要保留“暂停键压在人脸上”的视觉暴力。

### IMG_0005 / 女明星微妙的表情参考.png

Human 标签：未标注。

Human 备注：参考女明星的表情，无语、嫌弃的面部表情。

当前绑定问题：

- 这张现在绑在 MB_004 外卖骑手卡下，但内容实际属于“综艺素材近景 / 女明星反应”。
- 建议新增任务卡：`MB_009 女明星反应近景：无语、嫌弃、被剪成恶意证据`。

可学习：

- 女明星不是大恶人式表演，而是非常微小的无语、嫌弃、懒得解释。
- 这种表情在综艺语境里可能只是瞬间反应，但被剪辑后会变成“她在霸凌”的证据。

不要学：

- 不要把女明星拍成恶毒反派。
- 不要用过度夸张白眼或漫画式嫌弃。

转成镜头规则：

- 表情素材要短、轻、暧昧，足够被误读。
- Prompt 里要写“ambiguous micro-expression”，而不是“mean villain face”。

### IMG_0006 / 蛐蛐男明星.png

Human 标签：只要人物距离、只要站位。

Human 备注：男明星被其他人孤立蛐蛐后，表情不自然。属于许临的剪辑素材，他还会在上面 p 字恶意黑男明星。

可学习：

- 最关键的是前后景关系：被孤立者在前景，背后两三个人形成小团体。
- 被孤立者不需要被写成有错，他的“不自然”来自被观看、被排除、听到背后笑声后的身体僵硬。
- 这张适合作为“男明星有罪”第一轮剪辑的结构参考。

不要学：

- 不要把男明星设计成真实黑料携带者。
- 不要让镜头替网友定罪，应该让观众意识到“这个定罪是剪出来的”。

转成镜头规则：

- 前景男明星占画面一侧，背景小团体虚焦/半清晰。
- 恶意字幕由许临后期加，不让原始素材自己说死。

### IMG_0007 / 环境.png

Human 标签：只要影调、只要景别、只要空间。

Human 备注：父亲餐饮还会有他餐饮店所处的那种普通城市的早起劳作人民的一个环境侧写，让大家知道父亲只是个普通人。

可学习：

- 这是父亲餐饮线最重要的“世界感”：普通城市、早起、蓝灰调、街道、劳动。
- 父亲不是舆论事件主角，却被舆论波及；环境侧写能让这个损伤有重量。

不要学：

- 不要拍成苦情控诉。
- 不要用夸张差评弹幕砸向父亲。

转成镜头规则：

- 父亲线可以先有一两个空镜/环境镜头，再进入店内。
- 城市早晨的冷光、远处劳动者、小车/电动车，比大特写更有力。

### IMG_0008 / 外卖小哥在歇息.png

Human 标签：只要人物距离、只要空间。

Human 备注：在餐厅门口休息的玩手机的外卖小哥的景别。

可学习：

- 这张比“雨夜骑手看手机”更生活：不是戏剧化雨夜，而是门口、车上、等单、刷手机。
- 舆论进入的是劳动间隙，不需要搞成城市灾难场面。

不要学：

- 不要复制真实外卖品牌。
- 不要英雄化或消费骑手苦难。

转成镜头规则：

- 外卖骑手可以横向中远景，人物不大，手机只是小屏幕。
- 舆论视频可以通过声音/一点屏幕光进入，不一定需要清楚看到屏幕内容。

### IMG_0009 / 普通人在刷手机.png

Human 标签：只要景别、只要空间。

Human 备注：类似这种餐厅，大家都在吃瓜。

可学习：

- 这是“公共舆论蒙太奇”的核心：普通餐厅里的人边吃边刷，不是站出来审判。
- 人物松散分布，手机是低头动作的一部分。
- 这种空间能把娱乐事件和经济下行的日常疲惫连接起来。

不要学：

- 不要让所有人夸张围观同一个屏幕。
- 不要拍成社交媒体宣传片或科技批判海报。

转成镜头规则：

- 餐厅中景/中远景，背景有人低头刷手机，声音层面叠加热搜/短视频。
- 不要把吃瓜拍成整齐一致的群众行为，要保留普通人的散漫。

## 给后续 Agent 的指令

### 摄影 Agent

- 许临家：先空间中全景，再侧脸近景，电脑光反眼。
- 综艺素材：明亮广角固定机位建立群体关系，再切近景制造误读。
- 男明星被孤立：前景孤立者 + 背景小团体，保持暧昧，不替观众定罪。
- 父亲餐饮：先普通城市早晨环境，再进店内。
- 吃瓜蒙太奇：餐厅/门口/等待空间里的低头手机，不做宏大隐喻。

### 美术 Agent

- 许临房间：一线城市里弄/老旧小区，普通旧、不打理；电脑正常，可以是普通显示器或 MacBook，不刻意低配。
- 许临空间优先学《宇宙探索编辑部》的影调、生活质感和普通旧空间，不学极端破败。
- 综艺空间：干净、明亮、可消费、可截屏。
- 父亲线：普通城市与普通餐饮店，重在经营痕迹。
- 公共传播线：手机、电动车、餐桌、门口，不要未来屏幕墙。

### Prompt Agent

- 每条 prompt 必须引用 Human 标注维度：只要站位 / 只要空间 / 只要影调等。
- 对综艺文字、暂停键、恶意 P 字，不要求生成模型准确出字，改为后期合成。
- 负面约束必须加入：不要真实综艺 Logo、不要真实明星脸、不要复制参考片人物、不要复古 CRT、不要电影化贫穷、不要漏水苦难、不要把许临家拍成贫困奇观。
