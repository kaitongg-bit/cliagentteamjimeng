# 《定稿》Prompt Pack v2: Character Sheets and Keyshots

Agent: Prompt Structurer Agent  
Mode: 正式按 `prompt_structurer.yaml` 执行  
Human 纠偏: 人物设定提示词是“人物外貌设定图 / 白底三视图 / turn-around sheet”，不是剧情镜头里的角色描述。镜头提示词只保留关键镜头。

## A. 全局角色设定图规则

### 白底三视图基准

角色设定图使用纯白或浅灰白背景，正面、侧面、背面三视图并排。全身站姿，脚底对齐，比例真实，不做剧情动作，不加入房间、街道、综艺棚或任何故事环境。

### 角色一致性

每张角色图优先锁定：年龄感、脸型、发型、身形、姿态、服装层次、鞋、常用小道具和真实材质。表情保持中性或轻微疲惫，不做强情绪表演，避免一次图里出现多套完全不同服装。

### 服装与材质

服装必须可拍、可复用，偏中国当代城市真实穿着。材质要明确：洗旧棉、起球针织、普通西装料、廉价雨衣、反光条、围裙、旧夹克、运动裤、塑料扣件、磨损鞋底。避免真实品牌、真实 logo、可读长文字。

### 负面总约束

不要剧情环境，不要电影剧照布光，不要精致棚拍，不要动漫风，不要插画风，不要时装大片，不要夸张人设，不要真实明星脸，不要真实品牌，不要可读长文字，不要过度磨皮，不要把角色做成游戏皮肤。
