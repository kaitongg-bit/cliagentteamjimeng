# 《定稿》Prompt Pack v1

Agent: Prompt Structurer Agent  
Mode: 正式按 `prompt_structurer.yaml` 执行  
Scope: 不跑 CLI，不写 JSON；用于即梦图像/视频生成前的短提示词包。  
Human 最新偏好: 短结构、灰闷影调、低动态范围、轻微欠曝、普通相机压缩感；避免漂亮冷蓝剧照、品牌 logo、清晰屏幕文字。

## A. 全局影调规则

### 全片影像基准

近现实城市低成本纪实质感。画面要普通、拥挤、可拍，像廉价相机或普通手机在真实空间里记录到的生活现场。整体灰闷、脏白、冷灰，允许旧顶灯、楼道灯带一点微黄或微绿。低动态范围，轻微欠曝，夜景有普通相机噪点和视频压缩感。

### 光线规则

屏幕是光源、声源、反射源和压力源，不是信息展板。电脑光只轻轻照亮脸、手、桌面和墙面，不能变成漂亮蓝光。手机光短促、局部、催促。楼道灯硬、丑、微闪。综艺素材内部可以明亮商业，但进入许临房间后被压冷、压脏、压扁。

### 屏幕/UI 规则

屏幕文字不要求清晰。评论区、热搜、剪辑时间线、聊天框只作为遮挡、反射、声源和节奏。避免真实软件品牌、真实平台 logo、真实节目名、真实商标。需要观众读懂的信息只给项目名、问号变句号、发布成功、同步提示等极短瞬间。

### 全局负面约束

避免漂亮电影蓝光、赛博朋克、霓虹、广告摄影感、棚拍感、豪宅、loft、黑客工作站、动漫风、插画风、过度磨皮、真实 logo、清晰长文字、极端破败、漏水苦难、恐怖片破屋、把许临拍成忧郁男神或落魄天才。

## B. 人物设定提示词

### 许临

32 岁左右中国男性后期剪辑师，普通长相，长期熬夜，肩颈僵硬，反应慢半拍。穿深色旧 hoodie 或普通衬衫，头发不精心打理。表演是疲惫、安静、硬撑，不哭、不崩溃、不做痛苦大表情；他像一个被工作和舆论消耗到很钝的普通人。

### 李云帝

事业成功的中国男明星，主要以屏幕素材、综艺画面、暂停帧、广告二次传播出现。外形体面但不要神化，表情可被暂停和裁切误读。不能拍成天然反派，也不能拍成被圣化的受害者；他是被剪辑和网友判决的材料之一。

### 孟青桐

事业不顺的中国女演员，主要以综艺素材、广告、道歉相关舆论画面出现。表情要轻微、模糊、可被误读：低头、慢半拍、尴尬笑、看手机。不能拍成恶女，也不能拍成纯白无辜者；她同样是被剪辑和站队消耗的人。

### 外卖员 / 普通网友

普通劳动者和吃瓜者，出现在雨棚、楼道、通勤、门口等待空间。重点不是恶意，而是疲惫生活中的顺手转发、外放、评论和跟风。手机竖屏光盖住脸，表情不夸张，像日常间隙里被算法带着走的人。

### 甲方 / 平台 / 系统

不需要拟人化。用手机震动、语音弹窗、协作记录、发布后台、自动续播、同步提示表现。它们冷静、短促、持续催促，像比任何人都稳定运转的机器。

## C. 场景设定提示词

### 许临出租屋

一线城市里弄或老旧小区普通单间，15-20 平米，正常可居住但长期不打理。灰白旧墙、铝合金窗、旧家具、床或折叠沙发、现代电脑设备、充电线、移动硬盘、外卖盒、纸巾、文件自然堆积。旧、乱、闷，但不贫穷化，不漏水苦难。

### 电脑工作台 / 剪辑 UI

普通现代显示器或轻薄笔记本，深灰剪辑界面、模糊时间线、波形、素材缩略图、下载进度和项目名。画面重点是屏幕光、眼睛反光、手指停顿和窗口切换，不展示真实软件 logo，不让观众读长文字。

### 综艺素材空间

明亮、干净、可消费的综艺画面，圆桌、舞台灯、主持位、嘉宾笑声和暂停帧。它与出租屋的灰闷形成反差。素材可以被裁切、暂停、加字、降温，但不能让某一方天然显得有罪。

### 楼道 / 电梯

老旧小区楼道，硬白灯、微闪、墙根泛黄、湿脚印、门缝外放声。旧电梯门有划痕、胶印、小广告残留。商业广告亮色和手写字廉价共存，舆论像污损一样贴在公共空间表面。

### 小区雨棚 / 外卖公共空间

绿色或半透明雨棚，电动车、红绿灯、雨水、竖屏手机、外卖箱和湿地面。外卖员在劳动间隙刷视频，评论区盖住人脸。不要拍成宏大隐喻，只要普通城市夜雨里的顺手观看。

### 城市清晨

雨后灰白清晨，公交、站牌、湿路面、清洁车远声、车身广告被雨洗白。城市不浪漫，不宏大，只是系统继续运行。画面克制，不给救赎暖光。

### 父亲餐饮相关空间

小餐饮店或线上差评页面只作为现实后果一闪。父亲空间不要苦情：收银台、后厨、空桌、菜单、普通灯光。若出现页面，文字半虚，只表现差评流和平台压力，不展示真实店名或地址。
